# 公式核对完成确认与改进方案最终确定

## 一、已确认的基线公式汇总

### 1.1 SAES-SVD核心公式

#### CEALC目标函数（Equation 3）

$$argmin_{A_ℓ, B_ℓ} ||(A_ℓ·B_ℓ - W_ℓ)·X_ℓ||_F^2 + α_ℓ·||A_ℓ·B_ℓ·X_ℓ - W_ℓ·X_ℓ^f||_F^2$$



口头证明思路（3 句话）：

1. 二阶近似下：$ΔL≈12(⃗ΔW)TIF(⃗ΔW)\Delta L \approx \tfrac12 \vec(\Delta W)^T I_F \vec(\Delta W)ΔL≈21(ΔW)TIF(ΔW)$
2. 在 Fisher 空间中做 rank-r 截断 ⇔ 最小化 loss-weighted 误差
3. 因此在 Fisher 空间选择 β 比在 Frobenius 空间更合理





其中：

- 第一项：局部重构误差
- 第二项：全精度对齐误差
- $`α_ℓ ≥ 0`$：对齐强度系数

------

#### 等价变换（Equation 4-5）

定义中间变量：

$T_ℓ = W_ℓ · X_ℓ$          // 当前输入下的输出
$R_ℓ = W_ℓ · X_ℓ^f$        // 全精度输入$X_ℓ^f$的输出



等价形式（Equation 4）：

$||(A_ℓ·B_ℓ·X_ℓ - T_ℓ)||_F^2 + α_ℓ·||(A_ℓ·B_ℓ·X_ℓ - R_ℓ)||_F^2$
$= (1 + α_ℓ)·||A_ℓ·B_ℓ·X_ℓ - Z_ℓ||_F^2 + C_ℓ$ 



加权平均目标（Equation 5）：

$Z_ℓ = (T_ℓ + α_ℓ·R_ℓ) / (1 + α_ℓ) $
    $= W_ℓ · (X_ℓ + α_ℓ·X_ℓ^f) / (1 + α_ℓ)$



常数项：

$C_ℓ = ||T_ℓ||_F^2 + α_ℓ·||R_ℓ||_F^2 - (1+α_ℓ)·||Z_ℓ||_F^2$

------

#### 二阶统计量表示（Equation 9）

$Z_ℓ · X_ℓ^T = W_ℓ · (X_ℓ·X_ℓ^T + α_ℓ·X_ℓ^f·X_ℓ^T) / (1 + α_ℓ)$
            $= W_ℓ · (H_ℓ + β_ℓ·Δ_ℓ)$

其中：

$H_ℓ = X_ℓ · X_ℓ^T     $               // 输入协方差
$Δ_ℓ = (X_ℓ^f - X_ℓ) · X_ℓ^T       $   // 累积误差协方差
$β_ℓ = α_ℓ / (1 + α_ℓ) ∈ [0, 1)    $  // 归一化系数

注：$`Δ_ℓ = (X_ℓ^f - X_ℓ)·X_ℓ^T` $展开后等价于$ `X_ℓ^f·X_ℓ^T - X_ℓ·X_ℓ^T$`

------

#### Theorem 4.1（空间变换）

设 $`H_ℓ = X_ℓ·X_ℓ^T ≻ 0`（$正定），$`L_ℓ = H_ℓ^{-1/2}`$，则：

$argmin_{rank(U_ℓ)≤r_ℓ} ||U_ℓ·X_ℓ - Z_ℓ||_F^2 $
$= argmin_{rank(V_ℓ)≤r_ℓ} ||V_ℓ - M_ℓ·L_ℓ||_F^2$

其中：

$V_ℓ := U_ℓ · H_ℓ^{1/2}$
$M_ℓ = Z_ℓ · X_ℓ^T$

------

#### 最终优化目标（Equation 10）

$argmin_{A_ℓ, B_ℓ} ||A_ℓ·B_ℓ·H_ℓ^{1/2} - W_ℓ·(H_ℓ + β·Δ_ℓ)·H_ℓ^{-1/2}||_F^2$

定义目标矩阵：

$G_ℓ(β) = W_ℓ · (H_ℓ + β·Δ_ℓ) · H_ℓ^{-1/2}$

------

#### 闭式解（Equation 11-12）

截断SVD（Equation 11）：

$T_ℓ* = [G_ℓ]_{r_ℓ} = Ũ_{r_ℓ} · Σ_{r_ℓ} · Ṽ_{r_ℓ}^T$

低秩因子（Equation 12）：

$A_ℓ = Ũ_{r_ℓ} · Σ_{r_ℓ}^{1/2}$
$B_ℓ = Σ_{r_ℓ}^{1/2} · Ṽ_{r_ℓ}^T · L_ℓ$

其中$ `L_ℓ = H_ℓ^{-1/2}`$

最终压缩权重：

$W_ℓ' = A_ℓ · B_ℓ = Ũ_{r_ℓ} · Σ_{r_ℓ} · Ṽ_{r_ℓ}^T · H_ℓ^{-1/2}$

------

#### ACES优化目标（Section 4.2）

$β* = argmax_{β_ℓ ∈ [0,1]} ρ(β_ℓ)$

保留能量比（Retained Energy Ratio）：

$ρ(β_ℓ) = Σ_{i=1}^{r_ℓ} σ_i^2(G_ℓ(β_ℓ)) / Σ_{i=1}^{min(m_ℓ, n_ℓ)} σ_i^2(G_ℓ(β_ℓ))$
$       = 保留能量 / 总能量$
$       = RER$

------

### 1.2 GFWSVD核心公式

#### 经验Fisher信息（Equation 11）

$I_F(θ*) = E[g·g^T] = (1/|D|) · Σ_{i=1}^{|D|} g_i · g_i^T$

其中：

$g_i = vec(G_i)$
$G_i = ∇_W L(θ)|_{batch_i}    // 第i个batch的梯度$

------

#### Kronecker分解（Equation 12）

优化目标：

$min ||I_F - A ⊗ B||_F$

其中：

$A ∈ R^{n×n}$
$B ∈ R^{m×m}$
$⊗ 表示Kronecker积$

------

#### Theorem 1（GFWSVD核心定理）

**定理条件**：

设 $`W ∈ R^{n×m}` $为权重矩阵，若满足：

1. 损失函数来自MLE问题
2. 经验Fisher信息具有Kronecker积结构 $`I_F = A ⊗ B`$
3. 权重W服从矩阵变量正态分布$ `MVN(W*, B^{-1}, A^{-1})`$

**定理结论**：

最小化压缩后loss增量的最优rank-r近似为：

$Ŵ_r = L_B^{-T} · W̃_r · L_A^{-1}$

其中：

$A = L_A · L_A^T $       // Cholesky分解（注意方向）
$B = L_B · L_B^T $       // Cholesky分解
$W̃ = L_B^T · W* · L_A $  // 辅助矩阵
$W̃_r = SVD_r(W̃)  $      // rank-r截断SVD

------

#### 低秩因子形式（Equation 10）

$W_1 = sqrt(Ŝ_r) · V̂_r^T · L_A^{-1}    ∈ R^{r×m}$
$W_2 = L_B^{-T} · Û_r · sqrt(Ŝ_r)      ∈ R^{n×r}$

其中：

$Û_r · Ŝ_r · V̂_r^T = SVD_r(L_B^T · W* · L_A)$

------

#### Kronecker分解的高效计算（Equation 13-15）

置换后的Fisher矩阵：

$Ĩ_F = (1/|D|) · Σ_{i=1}^{|D|} (G_i ⊗ G_i)$

左乘向量的高效计算（Equation 15）：

$Ĩ_F · z = (1/|D|) · Σ_{i=1}^{|D|} vec(G_i^T · Z · G_i)$

其中$ `z = vec(Z), Z ∈ R^{n×n}`$

复杂度从 $`O(m^2 · n^2)` $降到$ `O(m·n^2 + m^2·n)`$

------

### 1.3 EoRA残差补偿公式

#### 目标

最小化输出误差：

$min ||ΔW · X||_F^2$

其中$ `ΔW = W - W'` $是压缩误差

------

#### 特征空间变换

对输入协方差做特征分解：

$X · X^T = Q · Λ · Q^T$

定义变换后的误差：

$ΔW' = ΔW · Q · Λ^{1/2}$

对 `ΔW'` 做SVD：

$ΔW' ≈ Ũ · Σ̃ · Ṽ^T$

------

#### 低秩因子定义

$B' = Ũ · Σ̃$
$A' = Ṽ^T$

------

#### 逆投影回原空间

$A = A' · Q'^{-1} = A' · Λ^{-1/2} · Q^T$

------

#### 最终前向传播

$W·X + B'·A·X$

------

#### 补偿矩阵

$W^{comp} = B' · A = Ũ_r · Σ̃_r · Ṽ_r^T · Λ^{-1/2} · Q^T$

------

#### 理论推导（验证等价性）

对于校准集的激活值 X（满足 $`X·X^T = Q·Λ·Q^T`$）：

$||ΔW·X - B·A·X||_F^2$
$= trace((ΔW - B·A)·X·X^T·(ΔW - B·A)^T)$
$= trace((ΔW - B·A)·Q·Λ·Q^T·(ΔW - B·A)^T)$
$= trace((ΔW·Q - B·A·Q)·Λ·(ΔW·Q - B·A·Q)^T)$
$= trace((ΔW·Q·Λ^{1/2} - B·A·Q·Λ^{1/2})·(...)^T)$
$= ||ΔW·Q·Λ^{1/2} - B·A·Q·Λ^{1/2}||_F^2$
$= ||ΔW' - B·A·Q'||_F^2$

------

## 二、改进方案最终确定

### 2.1 创新点一：Fisher-Guided β Selection

#### 原ACES目标（SAES-SVD）

在SAES-SVD中，β的选择通过最大化保留能量比（RER）来确定：

$β* = argmax_{β_ℓ ∈ [0,1]} ρ(β_ℓ)$

$ρ(β_ℓ) = Σ_{i=1}^{r_ℓ} σ_i^2(G_ℓ(β_ℓ)) / Σ_{i=1}^{min(m_ℓ, n_ℓ)} σ_i^2(G_ℓ(β_ℓ))$

其中：

$G_ℓ(β) = W_ℓ · (H_ℓ + β·Δ_ℓ) · H_ℓ^{-1/2}$



------

#### 问题分析

ACES仅基于数据统计量 $`H_ℓ` $和 $`Δ_ℓ`$ 选择β，未考虑不同参数对任务损失的敏感度差异：

- 对loss贡献大的参数方向与对loss贡献小的参数方向被同等对待
- β的选择与具体任务无关

------

#### 改进方案

引入GFWSVD的Fisher信息加权，将目标矩阵变换到Fisher度量空间。

**Step 1**：计算经验Fisher信息的Kronecker分解

$I_F ≈ A ⊗ B$

**Step 2**：对A和B进行Cholesky分解

$A = L_A · L_A^T$
$B = L_B · L_B^T$

**Step 3**：定义Fisher变换后的目标矩阵

$G_ℓ^F(β) = L_B^T · G_ℓ(β) · L_A$
$         = L_B^T · W_ℓ · (H_ℓ + β·Δ_ℓ) · H_ℓ^{-1/2} · L_A$

**Step 4**：在Fisher空间中优化β

$β* = argmax_{β_ℓ ∈ [0,1]} ρ^F(β_ℓ)$

$ρ^F(β_ℓ) = Σ_{i=1}^{r_ℓ} σ_i^2(G_ℓ^F(β_ℓ)) / Σ_{i=1}^{min(m_ℓ, n_ℓ)} σ_i^2(G_ℓ^F(β_ℓ))$

**Step 5**：使用最优β*进行压缩

// 在Fisher空间做截断SVD
$[G_ℓ^F(β*)]_{r_ℓ} = Û_{r_ℓ} · Ŝ_{r_ℓ} · V̂_{r_ℓ}^T$

// 映回原空间
$Ŵ_ℓ = L_B^{-T} · Û_{r_ℓ} · Ŝ_{r_ℓ} · V̂_{r_ℓ}^T · L_A^{-1} · H_ℓ^{-1/2}$

------

#### 理论依据

根据GFWSVD的Theorem 1，在Fisher度量下的低秩近似最小化loss增量：

$ΔL ≈ (1/2) · (θ - θ*)^T · I_F · (θ - θ*)$

因此在Fisher空间中选择β能够使压缩方案更好地保护对任务敏感的参数方向。

------

### 2.2 创新点二：Residual-Boosted Compression

#### Stage 1（CEALC主压缩）

使用创新点一的Fisher-Guided β Selection获得β*，然后进行CEALC压缩。

**构造目标矩阵**：

$G_ℓ(β*) = W_ℓ · (H_ℓ + β*·Δ_ℓ) · H_ℓ^{-1/2}$

**做rank-r1截断SVD**：

$G_ℓ(β*) ≈ Ũ_{r1} · Σ_{r1} · Ṽ_{r1}^T$

**Stage 1的压缩权重**：

$W_ℓ^{(1)} = Ũ_{r1} · Σ_{r1} · Ṽ_{r1}^T · H_ℓ^{-1/2}$

**对应的低秩因子**：

$A_ℓ^{(1)} = Ũ_{r1} · Σ_{r1}^{1/2}$
$B_ℓ^{(1)} = Σ_{r1}^{1/2} · Ṽ_{r1}^T · H_ℓ^{-1/2}$

------

#### Stage 2（特征空间残差补偿）

**Step 1**：计算输出误差

根据SAES-SVD的累积误差对齐思想，我们关心的是压缩后输出与全精度参考的输出差异：

$E_ℓ = W_ℓ · X_ℓ^f - W_ℓ^{(1)} · X_ℓ$



**Step 2**：特征空间变换（EoRA方式）

对输入协方差做特征分解：

$X_ℓ · X_ℓ^T = Q_ℓ · Λ_ℓ · Q_ℓ^T$

将误差变换到白化空间：

$E_ℓ' = E_ℓ · Q_ℓ · Λ_ℓ^{1/2}$



**Step 3**：残差SVD

对$E_ℓ'$做rank-r2截断SVD：

$E_ℓ' ≈ Ũ_{r2}' · Σ̃_{r2}' · Ṽ_{r2}'^T$



**Step 4**：逆投影回原空间

定义低秩因子：

$B_ℓ' = Ũ_{r2}' · Σ̃_{r2}'$
$A_ℓ' = Ṽ_{r2}'^T$

补偿矩阵：

$W_ℓ^{comp} = B_ℓ' · A_ℓ' · Λ_ℓ^{-1/2} · Q_ℓ^T$
          $ = Ũ_{r2}' · Σ̃_{r2}' · Ṽ_{r2}'^T · Λ_ℓ^{-1/2} · Q_ℓ^T$



**Step 5**：最终压缩权重

$W_ℓ^{(2)} = W_ℓ^{(1)} + W_ℓ^{comp}$

------

1. 前向传播，收集全精度激活 ${X_ℓ^f}_{ℓ=1}^L$

    // ─────────────────────────────────
    // Step 1: 逐层Fisher信息估计
    // ────────────────────────────────
4. $I_F^{(ℓ)}$ ← 0
5. for batch in D:
6. 计算第ℓ层梯度 $G_{batch} = ∇_{W_ℓ} L(batch)$
7. $g = vec(G_{batch})$
8. $I_F^{(ℓ)} ← I_F^{(ℓ)} + g · g^T$
9. $I_F^{(ℓ)} ← I_F^{(ℓ)} / |D|$

10. // Kronecker分解
11. $A_ℓ, B_ℓ = KroneckerDecompose(I_F^{(ℓ)})$

12. // Cholesky分解
13. $L_A^{(ℓ)} = chol(A_ℓ) $    // $A_ℓ = L_A^{(ℓ)} · L_A^{(ℓ)T}$
14. $L_B^{(ℓ)} = chol(B_ℓ)  $   // $B_ℓ = L_B^{(ℓ)} · L_B^{(ℓ)T}$

    // ─────────────────────────────────
    // Step 2: 统计量计算
    // ─────────────────────────────────
15. $H_ℓ = X_ℓ · X_ℓ^T$
16. $Δ_ℓ = (X_ℓ^f - X_ℓ) · X_ℓ^T$

    // ─────────────────────────────────
    // Step 3: Fisher-Guided β选择
    // ─────────────────────────────────
17. best_rho = 0, β* = 0
18. for β in {0, 0.1, 0.2, ..., 0.9}:
19. $G(β) = W_ℓ · (H_ℓ + β·Δ_ℓ) · H_ℓ^{-1/2}$
20. $G_F(β) = L_B^{(ℓ)T} · G(β) · L_A^{(ℓ)}$
21. 计算 $ρ^F(β)$ = (前r_ℓ个奇异值平方和) / (所有奇异值平方和)
22. if $ρ^F(β)$ > best_rho:
23. best_rho = $ρ^F(β)$, β* = β

    // ─────────────────────────────────
    // Step 4: Stage 1 - CEALC主压缩
    // ─────────────────────────────────
24. $G* = W_ℓ · (H_ℓ + β*·Δ_ℓ) · H_ℓ^{-1/2}$
25. $U, Σ, V^T = SVD(G*)$
26. $W_ℓ^{(1)} = U[:, :r1] · Σ[:r1, :r1] · V[:, :r1]^T · H_ℓ^{-1/2}$

    // ─────────────────────────────────
    // Step 5: Stage 2 - 残差补偿
    // ─────────────────────────────────
27. // 计算输出误差
28. $E_ℓ = W_ℓ · X_ℓ^f - W_ℓ^{(1)} · X_ℓ$

29. // 特征空间变换
30. $Q, Λ = eigh(X_ℓ · X_ℓ^T)$
31. $E_{white} = E_ℓ · Q · Λ^{1/2}$

32. // 残差SVD
33. $Ũ, Σ̃, Ṽ^T = SVD(E_white)$

34. // 补偿矩阵
35. $W_{comp} = Ũ[:, :r2] · Σ̃[:r2, :r2] · Ṽ[:r2, :]^T · Λ^{-1/2} · Q^T$

36. // 合并
37. $W_ℓ' = W_ℓ^{(1)} + W_{comp}$

    // ─────────────────────────────────
    // Step 6: 更新下游激活
    // ─────────────────────────────────
38. $X_{ℓ+1} = forward(W_ℓ', X_ℓ)$

39. return ${W_ℓ'}_{ℓ=1}^L$

```
9.     I_F^{(ℓ)} ← I_F^{(ℓ)} / |D|
    
10.    // Kronecker分解
11.    A_ℓ, B_ℓ = KroneckerDecompose(I_F^{(ℓ)})
    
12.    // Cholesky分解
13.    L_A^{(ℓ)} = chol(A_ℓ)     // A_ℓ = L_A^{(ℓ)} · L_A^{(ℓ)T}
14.    L_B^{(ℓ)} = chol(B_ℓ)     // B_ℓ = L_B^{(ℓ)} · L_B^{(ℓ)T}
    
    // ─────────────────────────────────
    // Step 2: 统计量计算
    // ─────────────────────────────────
15.    H_ℓ = X_ℓ · X_ℓ^T
16.    Δ_ℓ = (X_ℓ^f - X_ℓ) · X_ℓ^T
    
    // ─────────────────────────────────
    // Step 3: Fisher-Guided β选择
    // ─────────────────────────────────
17.    best_rho = 0, β* = 0
18.    for β in {0, 0.1, 0.2, ..., 0.9}:
19.        G(β) = W_ℓ · (H_ℓ + β·Δ_ℓ) · H_ℓ^{-1/2}
20.        G_F(β) = L_B^{(ℓ)T} · G(β) · L_A^{(ℓ)}
21.        计算 ρ^F(β) = (前r_ℓ个奇异值平方和) / (所有奇异值平方和)
22.        if ρ^F(β) > best_rho:
23.            best_rho = ρ^F(β), β* = β
    
    // ─────────────────────────────────
    // Step 4: Stage 1 - CEALC主压缩
    // ─────────────────────────────────
24.    G* = W_ℓ · (H_ℓ + β*·Δ_ℓ) · H_ℓ^{-1/2}
25.    U, Σ, V^T = SVD(G*)
26.    W_ℓ^{(1)} = U[:, :r1] · Σ[:r1, :r1] · V[:, :r1]^T · H_ℓ^{-1/2}
    
    // ─────────────────────────────────
    // Step 5: Stage 2 - 残差补偿
    // ─────────────────────────────────
27.    // 计算输出误差
28.    E_ℓ = W_ℓ · X_ℓ^f - W_ℓ^{(1)} · X_ℓ
    
29.    // 特征空间变换
30.    Q, Λ = eigh(X_ℓ · X_ℓ^T)
31.    E_white = E_ℓ · Q · Λ^{1/2}
    
32.    // 残差SVD
33.    Ũ, Σ̃, Ṽ^T = SVD(E_white)
    
34.    // 补偿矩阵
35.    W_comp = Ũ[:, :r2] · Σ̃[:r2, :r2] · Ṽ[:r2, :]^T · Λ^{-1/2} · Q^T
    
36.    // 合并
37.    W_ℓ' = W_ℓ^{(1)} + W_comp
    
    // ─────────────────────────────────
    // Step 6: 更新下游激活
    // ─────────────────────────────────
38.    X_{ℓ+1} = forward(W_ℓ', X_ℓ)

39. return {W_ℓ'}_{ℓ=1}^L
```

## 三、需要证明的理论

### 3.1 Theorem 1：Fisher加权截断最优性

**定理陈述**：

设 $`G_ℓ(β) = W_ℓ·(H_ℓ + β·Δ_ℓ)·H_ℓ^{-1/2}` $为CEALC的目标矩阵，$`I_F ≈ A ⊗ B` $为Fisher信息的Kronecker近似，$`L_A, L_B` $为对应的Cholesky因子。

定义Fisher变换后的目标矩阵：

$G_ℓ^F(β) = L_B^T · G_ℓ(β) · L_A$

则对 $`G_ℓ^F(β)`$ 做$rank_{r_ℓ}$截断SVD得到的解，在映回原空间后，相比于直接对 `G_ℓ(β)` 做截断SVD，能够获得更小的Fisher加权重构误差：

$||L_B^T · (W_ℓ - Ŵ_ℓ^F) · L_A||_F^2 ≤ ||L_B^T · (W_ℓ - Ŵ_ℓ) · L_A||_F^2$

其中 $`Ŵ_ℓ^F`$ 是Fisher引导的压缩结果，$`Ŵ_ℓ` $是原始CEALC的压缩结果。

**证明思路**：

1. 从GFWSVD的Theorem 1出发：Fisher加权误差 = $`||L_B^T·(W* - W)·L_A||_F^2`$
2. 证明在Fisher空间做SVD截断等价于最小化Fisher加权误差
3. 利用Eckart-Young-Mirsky定理确保最优性

------

### 3.2 Lemma 1：两阶段压缩优越性

**引理陈述**：

设CEALC Stage 1压缩得到$ W_ℓ^{(1)}$（rank = r1），Stage 2残差补偿得到 $W_ℓ^{comp}$（rank = r2），总秩预算 $r = r1 + r2$。

则两阶段压缩结果 $W_ℓ^{(2)} = W_ℓ^{(1)} + W_ℓ^{comp} $满足：

$||W_ℓ·X_ℓ^f - W_ℓ^{(2)}·X_ℓ||_F^2 ≤ ||W_ℓ·X_ℓ^f - W̃_ℓ·X_ℓ||_F^2$

其中 $W̃_ℓ $是对CEALC目标直接做rank-r截断的结果。

**证明思路**：

1. 利用ResSVD的核心不等式：残差的rank-r2近似不劣于丢弃的奇异值
2. 利用EoRA的特征空间变换：在输出误差空间做补偿
3. 结合三角不等式完成证明

### 3.3 Proposition 1：计算复杂度分析

**命题陈述**：

FR-SVD相对于SAES-SVD的额外计算开销为：

| ------------- | ------------------ | -------------------------------- |

### 问题3：Stage 1压缩输出形式

我写的是：

$W_ℓ^{(1)} = Ũ_{r1} · Σ_{r1} · Ṽ_{r1}^T · H_ℓ^{-1/2}$
