#  (一)Theorem 1 初步：Fisher加权截断最优性

## 一、定理陈述

### 1.1 定理内容

**Theorem 1（Fisher加权截断最优性）**

设：

- $`W_ℓ ∈ R^{m×n}` $为第ℓ层的原始权重矩阵
- $`H_ℓ = X_ℓ · X_ℓ^T ∈ R^{n×n}`$ 为输入协方差矩阵，且 `H_ℓ ≻ 0`（正定）
- $`Δ_ℓ = (X_ℓ^f - X_ℓ) · X_ℓ^T ∈ R^{n×n}`$ 为累积误差协方差
- $`β ∈ [0, 1)` $为累积误差补偿系数
- $`I_F^{(ℓ)} ≈ A_ℓ ⊗ B_ℓ` $为第ℓ层经验Fisher信息的Kronecker近似
- $`A_ℓ = L_A · L_A^T`$，$`B_ℓ = L_B · L_B^T`$ 为Cholesky分解

定义CEALC目标矩阵：

$G_ℓ(β) = W_ℓ · (H_ℓ + β·Δ_ℓ) · H_ℓ^{-1/2}$



定义Fisher变换后的目标矩阵：

$G_ℓ^F(β) = L_B^T · G_ℓ(β) · L_A$



**方法1（原始CEALC）**：

$T_ℓ = [G_ℓ(β)]_{r_ℓ}     $                      // 普通空间截断SVD
$Ŵ_ℓ = T_ℓ · H_ℓ^{-1/2}   $                      // 映回权重空间



**方法2（Fisher-Guided CEALC）**：

$T_ℓ^F = [G_ℓ^F(β)]_{r_ℓ}$                       // Fisher空间截断SVD
$Ŵ_ℓ^F = L_B^{-T} · T_ℓ^F · L_A^{-1} · H_ℓ^{-1/2}$   // 映回权重空间



**结论**：

$||L_B^T · (W_ℓ - Ŵ_ℓ^F) · L_A||_F^2 ≤ ||L_B^T · (W_ℓ - Ŵ_ℓ) · L_A||_F^2$

即：在Fisher加权误差意义下，方法2不劣于方法1。



$||L_B^T · (W_ℓ - Ŵ_ℓ^F) · L_A||_F^2近似压缩前后的损失增量||W_ℓ - Ŵ_ℓ||_F^2$

## 二、预备知识

### 2.1 Eckart-Young-Mirsky定理

**来源**：Eckart & Young (1936), "The approximation of one matrix by another of lower rank"

**定理内容**：

对于任意矩阵 `M ∈ R^{m×n}`，其在Frobenius范数意义下的最优rank-r近似为截断SVD：

$[M]_r = argmin_{rank(N)≤r} ||M - N||_F^2$ 

设 `M = U·Σ·V^T` 为M的SVD，其中 `Σ = diag(σ_1, σ_2, ..., σ_p)`，`p = min(m,n)`，`σ_1 ≥ σ_2 ≥ ... ≥ σ_p ≥ 0`。



则：

$[M]_r = U_r · Σ_r · V_r^T$

其中 `U_r, V_r` 为前r列，`Σ_r = diag(σ_1, ..., σ_r)`。



最优误差为：

$||M - [M]_r||_F^2 = Σ_{i=r+1}^{p} σ_i^2$ 

------

### 2.2 Frobenius范数的性质

**来源**：标准线性代数教材

**性质1（定义）**：

$||M||_F^2 = Σ_{i,j} M_{ij}^2 = trace(M^T · M) = trace(M · M^T)$ 



**性质2（正交不变性）**：

对于正交矩阵 `Q`（满足 `Q^T·Q = I`）：

$||Q · M||_F = ||M · Q||_F = ||M||_F$  



**性质3（与奇异值的关系）**：

$||M||_F^2 = Σ_{i=1}^{p} σ_i^2(M)$



------

### 2.3 GFWSVD的核心结论

**来源**：GFWSVD论文 Theorem 1 及 Equation 9

**结论**：在Fisher信息 `I_F = A ⊗ B` 的度量下，压缩误差可表示为：

$vec(W - Ŵ)^T · I_F · vec(W - Ŵ) = ||L_B^T · (W - Ŵ) · L_A||_F^2$



**推导**（GFWSVD论文 Equation 9）：

由Kronecker积的性质：

$vec(W - Ŵ)^T · (A ⊗ B) · vec(W - Ŵ)$
$= vec(W - Ŵ)^T · (L_A·L_A^T ⊗ L_B·L_B^T) · vec(W - Ŵ)$



利用 `(A ⊗ B) = (L_A ⊗ L_B) · (L_A^T ⊗ L_B^T)`：

$= vec(W - Ŵ)^T · (L_A ⊗ L_B) · (L_A^T ⊗ L_B^T) · vec(W - Ŵ)$



利用Kronecker(克罗内克)积与vec的关系 `(A ⊗ B) · vec(C) = vec(B·C·A^T)`：

$(L_A^T ⊗ L_B^T) · vec(W - Ŵ) = vec(L_B^T · (W - Ŵ) · L_A)$

因此：

$= vec(L_B^T · (W - Ŵ) · L_A)^T · vec(L_B^T · (W - Ŵ) · L_A)$
$= ||L_B^T · (W - Ŵ) · L_A||_F^2$



------

### 2.4 SAES-SVD的CEALC框架

**来源**：SAES-SVD论文 Equation 10-12

**CEALC目标函数**（Equation 10）：

$argmin_{A_ℓ, B_ℓ} ||A_ℓ·B_ℓ·H_ℓ^{1/2} - W_ℓ·(H_ℓ + β·Δ_ℓ)·H_ℓ^{-1/2}||_F^2$



**目标矩阵定义**：

$G_ℓ(β) = W_ℓ · (H_ℓ + β·Δ_ℓ) · H_ℓ^{-1/2}$



**闭式解**（Equation 11-12）：

$[G_ℓ(β)]_{r_ℓ} = Ũ_{r_ℓ} · Σ_{r_ℓ} · Ṽ_{r_ℓ}^T$
$A_ℓ = Ũ_{r_ℓ} · Σ_{r_ℓ}^{1/2}$
$B_ℓ = Σ_{r_ℓ}^{1/2} · Ṽ_{r_ℓ}^T · H_ℓ^{-1/2}$



**压缩权重**：

$Ŵ_ℓ = A_ℓ · B_ℓ = Ũ_{r_ℓ} · Σ_{r_ℓ} · Ṽ_{r_ℓ}^T · H_ℓ^{-1/2}$
$    = [G_ℓ(β)]_{r_ℓ} · H_ℓ^{-1/2}$

## 三、主定理证明

### 3.1 第一步：定义Fisher加权重构误差

我们要比较的目标是**Fisher加权重构误差**，定义为：

$E_F(Ŵ) := ||L_B^T · (W_ℓ - Ŵ) · L_A||_F^2$ 

**来源**：GFWSVD论文 Equation 9（见预备知识2.3）

这个度量的意义是：它近似于压缩导致的loss增量。



------

### 3.2 第二步：分析原始CEALC方法的Fisher误差

**原始CEALC方法**产生的压缩权重为：

$Ŵ_ℓ = [G_ℓ(β)]_{r_ℓ} · H_ℓ^{-1/2}$



其Fisher加权误差为：

$E_F(Ŵ_ℓ) = ||L_B^T · (W_ℓ - Ŵ_ℓ) · L_A||_F^2$



将 $`Ŵ_ℓ = [G_ℓ(β)]_{r_ℓ} · H_ℓ^{-1/2}` $代入：



$W_ℓ - Ŵ_ℓ = W_ℓ - [G_ℓ(β)]_{r_ℓ} · H_ℓ^{-1/2}$



注意到：

$G_ℓ(β) · H_ℓ^{-1/2} = W_ℓ · (H_ℓ + β·Δ_ℓ) · H_ℓ^{-1/2} · H_ℓ^{-1/2}$
$                    = W_ℓ · (H_ℓ + β·Δ_ℓ) · H_ℓ^{-1}$



这个形式比较复杂。为了简化分析，我们引入一个辅助变量。



------

### 3.3 第三步：引入辅助变量简化分析

定义辅助矩阵：

$M_ℓ = G_ℓ(β) = W_ℓ · (H_ℓ + β·Δ_ℓ) · H_ℓ^{-1/2}$



原始CEALC的压缩结果可以写成：

$Ŵ_ℓ = [M_ℓ]_{r_ℓ} · H_ℓ^{-1/2}$



为了分析Fisher误差，我们需要计算：

$L_B^T · (W_ℓ - Ŵ_ℓ) · L_A$



**引理3.1**：定义 $`W̃_ℓ = W_ℓ · H_ℓ^{1/2}`$，则：

$W_ℓ = W̃_ℓ · H_ℓ^{-1/2}$
$Ŵ_ℓ = [M_ℓ]_{r_ℓ} · H_ℓ^{-1/2}$



因此：

$W_ℓ - Ŵ_ℓ = (W̃_ℓ - [M_ℓ]_{r_ℓ}) · H_ℓ^{-1/2}$



------

### 3.4 第四步：计算原始CEALC的Fisher误差

将上述结果代入Fisher误差表达式：

$E_F(Ŵ_ℓ) = ||L_B^T · (W_ℓ - Ŵ_ℓ) · L_A||_F^2$
$         = ||L_B^T · (W̃_ℓ - [M_ℓ]_{r_ℓ}) · H_ℓ^{-1/2} · L_A||_F^2$



定义：

$P := H_ℓ^{-1/2} · L_A$



则：

$E_F(Ŵ_ℓ) = ||L_B^T · (W̃_ℓ - [M_ℓ]_{r_ℓ}) · P||_F^2$



**注意**：$`W̃_ℓ = W_ℓ · H_ℓ^{1/2}`$，而 $`M_ℓ = W_ℓ · (H_ℓ + β·Δ_ℓ) · H_ℓ^{-1/2}`$



当 `β = 0` 时：

$M_ℓ|_{β=0} = W_ℓ · H_ℓ · H_ℓ^{-1/2} = W_ℓ · H_ℓ^{1/2} = W̃_ℓ$



但一般情况下 `β ≠ 0`，所以 $`W̃_ℓ ≠ M_ℓ`$。



------

### 3.5 第五步：分析Fisher-Guided CEALC方法

**Fisher-Guided方法**首先在Fisher空间定义目标：

$G_ℓ^F(β) = L_B^T · G_ℓ(β) · L_A$
$         = L_B^T · M_ℓ · L_A$ 



在Fisher空间做截断SVD：

$[G_ℓ^F(β)]_{r_ℓ} = Û_{r_ℓ} · Ŝ_{r_ℓ} · V̂_{r_ℓ}^T$

然后映回原空间。映回过程需要仔细分析。



------

### 3.6 第六步：Fisher-Guided方法的映回过程

我们需要找到一个权重矩阵 $`Ŵ_ℓ^F`$，使得在CEALC框架下它对应于Fisher空间的截断结果。

**关键观察**：

在CEALC框架中，压缩权重 `Ŵ` 与目标矩阵 `T` 的关系是：

$Ŵ = T · H_ℓ^{-1/2}$

其中 $`T = A·B·H_ℓ^{1/2}`$（见SAES-SVD Equation 10）。



对于Fisher-Guided方法，我们在Fisher空间得到了截断结果：

$T_ℓ^F = [G_ℓ^F(β)]_{r_ℓ} = [L_B^T · M_ℓ · L_A]_{r_ℓ}$



要映回原空间，需要求解：

$L_B^T · T · L_A = T_ℓ^F$

解得：

$T = L_B^{-T} · T_ℓ^F · L_A^{-1}$

因此Fisher-Guided方法的压缩权重为：

$Ŵ_ℓ^F = T · H_ℓ^{-1/2}$
       $= L_B^{-T} · T_ℓ^F · L_A^{-1} · H_ℓ^{-1/2}$

​       $= L_B^{-T} · [L_B^T · M_ℓ · L_A]_{r_ℓ} · L_A^{-1} · H_ℓ^{-1/2} $

 

------

### 3.7 第七步：计算Fisher-Guided方法的Fisher误差

Fisher-Guided方法的Fisher加权误差为：

$E_F(Ŵ_ℓ^F) = ||L_B^T · (W_ℓ - Ŵ_ℓ^F) · L_A||_F^2$



代入 $`Ŵ_ℓ^F`$：

$W_ℓ - Ŵ_ℓ^F = W_ℓ - L_B^{-T} · [L_B^T · M_ℓ · L_A]_{r_ℓ} · L_A^{-1} · H_ℓ^{-1/2}$



左乘 $`L_B^T`$，右乘 $`L_A`$：

$L_B^T · (W_ℓ - Ŵ_ℓ^F) · L_A = L_B^T · W_ℓ · L_A - L_B^T · L_B^{-T} · [L_B^T · M_ℓ · L_A]_{r_ℓ} · L_A^{-1} · H_ℓ^{-1/2} · L_A$
                                       $= L_B^T · W_ℓ · L_A - [L_B^T · M_ℓ · L_A]_{r_ℓ} · L_A^{-1} · H_ℓ^{-1/2} · L_A $ 

注意到 $`L_B^T · L_B^{-T} = I`$。



定义：

$Q := L_A^{-1} · H_ℓ^{-1/2} · L_A$



则：

$L_B^T · (W_ℓ - Ŵ_ℓ^F) · L_A = L_B^T · W_ℓ · L_A - [L_B^T · M_ℓ · L_A]_{r_ℓ} · Q$



------

### 3.8 第八步：简化分析——在β=0的特殊情况

为了使证明更清晰，我们先考虑 `β = 0` 的特殊情况。

当 `β = 0` 时：

$M_ℓ = G_ℓ(0) = W_ℓ · H_ℓ · H_ℓ^{-1/2} = W_ℓ · H_ℓ^{1/2}$



此时：

$G_ℓ^F(0) = L_B^T · M_ℓ · L_A = L_B^T · W_ℓ · H_ℓ^{1/2} · L_A$



**原始CEALC的Fisher误差**（β=0时）：

压缩权重：

$Ŵ_ℓ = [M_ℓ]_{r_ℓ} · H_ℓ^{-1/2} = [W_ℓ · H_ℓ^{1/2}]_{r_ℓ} · H_ℓ^{-1/2}$



Fisher误差：

$E_F(Ŵ_ℓ) = ||L_B^T · (W_ℓ - [W_ℓ · H_ℓ^{1/2}]_{r_ℓ} · H_ℓ^{-1/2}) · L_A||_F^2$



令 $`W̃ = W_ℓ · H_ℓ^{1/2}`$，则 $`W_ℓ = W̃ · H_ℓ^{-1/2}`$，

因此：

$W_ℓ - [W̃]_{r_ℓ} · H_ℓ^{-1/2} = (W̃ - [W̃]_{r_ℓ}) · H_ℓ^{-1/2}$



所以：

$E_F(Ŵ_ℓ) = ||L_B^T · (W̃ - [W̃]_{r_ℓ}) · H_ℓ^{-1/2} · L_A||_F^2$



------

**Fisher-Guided方法的Fisher误差**（β=0时）：

Fisher空间的目标矩阵：

$G_ℓ^F(0) = L_B^T · W̃ · L_A$ 



Fisher空间的截断结果：

$T_ℓ^F = [L_B^T · W̃ · L_A]_{r_ℓ}$



映回权重空间：

$Ŵ_ℓ^F = L_B^{-T} · T_ℓ^F · L_A^{-1} · H_ℓ^{-1/2}$ 



Fisher误差：

$E_F(Ŵ_ℓ^F) = ||L_B^T · (W_ℓ - Ŵ_ℓ^F) · L_A||_F^2$ 



代入：

$L_B^T · W_ℓ · L_A = L_B^T · W̃ · H_ℓ^{-1/2} · L_A$



$L_B^T · Ŵ_ℓ^F · L_A = L_B^T · L_B^{-T} · T_ℓ^F · L_A^{-1} · H_ℓ^{-1/2} · L_A$
                         $= T_ℓ^F · L_A^{-1} · H_ℓ^{-1/2} · L_A$

 

定义 $`R := L_A^{-1} · H_ℓ^{-1/2} · L_A`$，则：

$L_B^T · (W_ℓ - Ŵ_ℓ^F) · L_A = L_B^T · W̃ · H_ℓ^{-1/2} · L_A - T_ℓ^F · R$
                                       $= L_B^T · W̃ · L_A · R - T_ℓ^F · R$
                                       $= (L_B^T · W̃ · L_A - T_ℓ^F) · R$



注意到 $`T_ℓ^F = [L_B^T · W̃ · L_A]_{r_ℓ}`$，所以：

$L_B^T · W̃ · L_A - T_ℓ^F = L_B^T · W̃ · L_A - [L_B^T · W̃ · L_A]_{r_ℓ}$ 

这是矩阵 $`L_B^T · W̃ · L_A` $的**截断残差**。

### 3.10 第十步：重新整理证明（更严格的版本）

让我采用一个更直接的证明方法。

**定义**：

$N = L_B^T · W̃ · L_A$    // Fisher变换后的辅助矩阵，其中$ W̃ = W_ℓ · H_ℓ^{1/2}$



**Fisher-Guided方法**在Fisher空间对N做截断：

$[N]_{r_ℓ} = [L_B^T · W̃ · L_A]_{r_ℓ}$



**关键引理**：

**引理3.2**：设 `N ∈ R^{m×n}`，`[N]_r` 是其rank-r最优近似。则对于任意rank-r矩阵 `M`：

$||N - [N]_r||_F^2 ≤ ||N - M||_F^2$



**来源**：Eckart-Young-Mirsky定理（预备知识2.1）



------

**应用引理3.2**：

设 $`N = L_B^T · W̃ · L_A`$。



**Fisher-Guided方法**产生的截断是 $`[N]_{r_ℓ}`$，其误差为：

$||N - [N]_{r_ℓ}||_F^2 = Σ_{i=r_ℓ+1}^{p} σ_i^2(N)$

其中$ `σ_i(N)` $是N的第i个奇异值。



**原始CEALC方法**产生的对应矩阵是什么？

原始CEALC在普通空间对 `W̃` 做截断：

$[W̃]_{r_ℓ} = Ũ_{r_ℓ} · Σ̃_{r_ℓ} · Ṽ_{r_ℓ}^T$



其对应的Fisher空间矩阵是：

$L_B^T · [W̃]_{r_ℓ} · L_A$



这是一个rank-r_ℓ矩阵（因为rank在线性变换下不增加）。

根据引理3.2：

$||N - [N]_{r_ℓ}||_F^2 ≤ ||N - L_B^T · [W̃]_{r_ℓ} · L_A||_F^2$



即：

$||L_B^T · W̃ · L_A - [L_B^T · W̃ · L_A]_{r_ℓ}||_F^2 $$≤ ||L_B^T · W̃ · L_A - L_B^T · [W̃]_{r_ℓ} · L_A||_F^2$

​                                                                 $= ||L_B^T · (W̃ - [W̃]_{r_ℓ}) · L_A||_F^2 $

### 3.11 第十一步：从辅助矩阵回到权重矩阵

回顾 $`W̃ = W_ℓ · H_ℓ^{1/2}`$，所以 $`W_ℓ = W̃ · H_ℓ^{-1/2}`$。



**原始CEALC的压缩权重**（β=0时）：

$Ŵ_ℓ = [W̃]_{r_ℓ} · H_ℓ^{-1/2}$



**原始CEALC的Fisher误差**：

$E_F(Ŵ_ℓ) = ||L_B^T · (W_ℓ - Ŵ_ℓ) · L_A||_F^2$
               $= ||L_B^T · (W̃ · H_ℓ^{-1/2} - [W̃]_{r_ℓ} · H_ℓ^{-1/2}) · L_A||_F^2$
               $= ||L_B^T · (W̃ - [W̃]_{r_ℓ}) · H_ℓ^{-1/2} · L_A||_F^2$



**Fisher-Guided方法的压缩权重**（β=0时）：

需要从Fisher空间的截断结果 $`[N]_{r_ℓ}` $恢复权重。

设 $`[N]_{r_ℓ} = Û_{r_ℓ} · Ŝ_{r_ℓ} · V̂_{r_ℓ}^T`$，则对应的 `W̃` 空间矩阵为：

$W̃^F = L_B^{-T} · [N]_{r_ℓ} · L_A^{-1}$



压缩权重为：

$Ŵ_ℓ^F = W̃^F · H_ℓ^{-1/2} = L_B^{-T} · [N]_{r_ℓ} · L_A^{-1} · H_ℓ^{-1/2}$



**Fisher-Guided方法的Fisher误差**：

$E_F(Ŵ_ℓ^F) = ||L_B^T · (W_ℓ - Ŵ_ℓ^F) · L_A||_F^2$
                $= ||L_B^T · (W̃ · H_ℓ^{-1/2} - L_B^{-T} · [N]_{r_ℓ} · L_A^{-1} · H_ℓ^{-1/2}) · L_A||_F^2$

展开：

$L_B^T · W̃ · H_ℓ^{-1/2} · L_A - L_B^T · L_B^{-T} · [N]_{r_ℓ} · L_A^{-1} · H_ℓ^{-1/2} · L_A$
$= L_B^T · W̃ · H_ℓ^{-1/2} · L_A - [N]_{r_ℓ} · L_A^{-1} · H_ℓ^{-1/2} · L_A$ 



定义 $`R = L_A^{-1} · H_ℓ^{-1/2} · L_A`$，则：

$= L_B^T · W̃ · L_A · R - [N]_{r_ℓ} · R$
$= (L_B^T · W̃ · L_A - [N]_{r_ℓ}) · R$
$= (N - [N]_{r_ℓ}) · R$



因此：

$E_F(Ŵ_ℓ^F) = ||(N - [N]_{r_ℓ}) · R||_F^2$

### 3.12 第十二步：比较两种方法的Fisher误差

我们现在有：

$E_F(Ŵ_ℓ) = ||L_B^T · (W̃ - [W̃]_{r_ℓ}) · H_ℓ^{-1/2} · L_A||_F^2$

其中 $`N = L_B^T · W̃ · L_A`$，$`R = L_A^{-1} · H_ℓ^{-1/2} · L_A`$。

### 3.13 第十三步：简化的核心定理（充分条件版本）

$E_F(Ŵ_ℓ^F) ≤ E_F(Ŵ_ℓ)$

### 4.1 预备引理

**Lemma 1（Eckart-Young-Mirsky定理）**

**来源**：Eckart & Young (1936), "The approximation of one matrix by another of lower rank"

**陈述**：对于任意矩阵 $`M ∈ R^{m×n}`$，其在Frobenius范数意义下的最优rank-r近似为截断SVD：

Code

$[M]_r = argmin_{rank(N)≤r} ||M - N||_F^2$

设 $`M = U · Σ · V^T`$ 为M的完整SVD，其中 $`Σ = diag(σ_1, σ_2, ..., σ_p)`$，$`p = min(m,n)`$，$`σ_1 ≥ σ_2 ≥ ... ≥ σ_p ≥ 0`$。

则：

$[M]_r = U_r · Σ_r · V_r^T$

最优误差为：

$||M - [M]_r||_F^2 = Σ_{i=r+1}^{p} σ_i^2$

------

**Lemma 2（可逆线性变换下秩不变）**

**来源**：标准线性代数

**陈述**：设 $`P ∈ R^{m×m}`, `Q ∈ R^{n×n}` $均可逆，$`M ∈ R^{m×n}`$，则：

$rank(P · M · Q) = rank(M)$

**证明**：

$rank(P · M · Q) ≤ rank(M · Q) ≤ rank(M)$           // 秩不增性
$rank(M) = rank(P^{-1} · (P · M · Q) · Q^{-1}) ≤ rank(P · M · Q)$  // 反向不等式

因此 $`rank(P · M · Q) = rank(M)`$。■

------

**Lemma 3（变量替换的等价性）**

**陈述**：设 `P, Q` 可逆，定义变量替换 $`S = P · T · Q`$。则优化问题

$min_{rank(T)≤r} ||P · (G - T) · Q||_F^2$

等价于

$min_{rank(S)≤r} ||P · G · Q - S||_F^2$

**证明**：

由 `S = P · T · Q` 且 `P, Q` 可逆，有 `T = P^{-1} · S · Q^{-1}`。

由Lemma 2：`rank(S) = rank(T)`。

因此 `rank(T) ≤ r` 当且仅当 `rank(S) ≤ r`。

目标函数变换：

$||P · (G - T) · Q||_F^2 = ||P · G · Q - P · T · Q||_F^2 = ||P · G · Q - S||_F^2$

两个优化问题的可行域和目标函数完全对应，因此等价。■

$min_{rank(S)≤r} ||N - S||_F^2$

**Step 3：应用Eckart-Young-Mirsky定理**

由Lemma 1（Eckart-Young-Mirsky定理），该问题的全局最优解为：

$S* = [N]_r = [L_B^T · G(β) · L_A]_r$

最优目标值为：

$||N - S*||_F^2 = ||N - [N]_r||_F^2 = Σ_{i=r+1}^{min(m,n)} σ_i^2(N)$

其中 $`σ_i(N)`$ 是 `N` 的第 `i` 个奇异值。

**Step 4：映回原变量**

由变量替换 $`S = L_B^T · T · L_A`$，最优解 `S*` 对应的 `T*` 为：

$T* = L_B^{-T} · S* · L_A^{-1}$
   $= L_B^{-T} · [L_B^T · G(β) · L_A]_r · L_A^{-1}$

**Step 5：验证秩约束**

由Lemma 2：

$rank(T*) = rank(L_B^{-T} · S* · L_A^{-1}) = rank(S*) = rank([N]_r) ≤ r$

因此 `T*` 满足秩约束。

**Step 6：证明结论(1)**

综合Step 1-5，`T*` 是目标函数 `J(T)` 在约束 `rank(T) ≤ r` 下的全局最优解。

结论(1)证毕。■

------

**Step 7：证明结论(2)**

原始CEALC方法的解为：

$T_0 = [G(β)]_r$

我们需要证明：

$J(T*) ≤ J(T_0)$

即：

$||L_B^T · (G(β) - T*) · L_A||_F^2 ≤ ||L_B^T · (G(β) - T_0) · L_A||_F^2$

**验证T_0的可行性**：

由截断SVD的性质：$`rank(T_0) = rank([G(β)]_r) ≤ r`$。

因此 `T_0` 是优化问题的一个可行解。

**应用最优性**：

由Step 6，`T*` 是全局最优解，因此对任意可行解（包括 `T_0`）：

$J(T*) ≤ J(T_0)$

结论(2)证毕。■

------

**Step 8：展开不等式**

将 `J(T*)` 和 `J(T_0)` 展开：

$J(T*) = ||L_B^T · (G(β) - T*) · L_A||_F^2$
      $= ||L_B^T · G(β) · L_A - L_B^T · T* · L_A||_F^2$
      $= ||N - S*||_F^2$
      $= ||N - [N]_r||_F^2$



$J(T_0) = ||L_B^T · (G(β) - [G(β)]_r) · L_A||_F^2$
       $= ||N - L_B^T · [G(β)]_r · L_A||_F^2$

定义 $`M_0 := L_B^T · [G(β)]_r · L_A`$。

由Lemma 2：$`rank(M_0) = rank([G(β)]_r) ≤ r`$。

因此 `M_0` 是一个rank不超过r的矩阵。

由Eckart-Young-Mirsky定理，对任意rank不超过r的矩阵M：

$||N - [N]_r||_F^2 ≤ ||N - M||_F^2$

特别地，取 `M = M_0`：

$||N - [N]_r||_F^2 ≤ ||N - M_0||_F^2$

即：

$J(T*) ≤ J(T_0)$

结论(2)的另一种证明完成。■

------

**Step 9：证明结论(3)**

结论(3)是结论(2)的直接推论。

不等式 `J(T*) ≤ J(T_0)` 说明：

- `T*` 在Fisher加权Frobenius范数 $`||L_B^T · (·) · L_A||_F` $意义下
- 是 `G(β)` 的更优（或同样优）的rank-r近似
- 相比于原始CEALC方法在普通Frobenius范数下得到的$ `T_0 = [G(β)]_r`​$

结论(3)证毕。■

------

### 4.3 定理证明完成

**Theorem 1 完整证明完成。**■

## 五、重要说明与讨论

### 5.1 定理证明了什么

**明确结论**：

Theorem 1证明的是：**在Fisher加权意义下，`T\*` 是目标矩阵 `G(β)` 的最优rank-r近似。**

具体地：

$||L_B^T · (G(β) - T*) · L_A||_F^2 ≤ ||L_B^T · (G(β) - T_0) · L_A||_F^2$

其中：

- $`T* = L_B^{-T} · [L_B^T · G(β) · L_A]_r · L_A^{-1}`$（Fisher-Guided方法）
- $`T_0 = [G(β)]_r`​$（原始CEALC方法）

### 5.2 定理没有直接证明什么

**需要明确的边界**：

Theorem 1 **没有** 直接证明：

$||L_B^T · (W - Ŵ^F) · L_A||_F^2 ≤ ||L_B^T · (W - Ŵ) · L_A||_F^2$

其中 $`Ŵ^F = T* · H^{-1/2}`, `Ŵ = T_0 · H^{-1/2}`$。

**原因**：

从 `G(β)` 到 `W` 需要右乘 $`H^{-1/2}`$。设压缩权重为 $`Ŵ = T · H^{-1/2}`$，则权重Fisher误差为：

$||L_B^T · (W - Ŵ) · L_A||_F^2 = ||L_B^T · (W - T · H^{-1/2}) · L_A||_F^2$

这等价于优化问题：

$min_{rank(T)≤r} ||L_B^T · W · L_A - L_B^T · T · H^{-1/2} · L_A||_F^2$

定义 `R = H^{-1/2} · L_A`，则问题变为：

$min_{rank(T)≤r} ||L_B^T · W · L_A - L_B^T · T · R||_F^2$

这是一个**加权低秩逼近问题**，其最优解一般不是对 $`L_B^T · G(β) · L_A` $做截断SVD。

### 5.3 定理的实践意义

虽然Theorem 1没有直接证明权重Fisher误差的最优性，但它仍然具有重要的实践意义：

**理由1：G(β)与loss增量的关系**

根据SAES-SVD论文Equation 3-10的推导：

$L_CEALC = ||A·B·X - W·X||_F^2 + α·||A·B·X - W·X^f||_F^2$
        $∝ ||A·B·H^{1/2} - G(β)||_F^2$

因此，`G(β)` 的重构误差直接关联到CEALC的目标函数（局部重构+累积误差对齐）。

**理由2：Fisher加权的意义**

Fisher加权 $`||L_B^T · (·) · L_A||_F`$ 近似于参数扰动导致的loss变化：

$ΔL ≈ (1/2) · vec(δW)^T · I_F · vec(δW) = (1/2) · ||L_B^T · δW · L_A||_F^2$

因此，最小化 `G(β)` 的Fisher加权重构误差，间接地有助于减小压缩导致的loss增量。

**理由3：实验验证**

定理的实践效果需要通过实验验证。理论保证的是"在G空间的Fisher加权意义下最优"，实验可以验证这是否转化为更好的下游任务性能。

### 5.4 如果要证明权重Fisher误差的严格最优性

若坚持要证明权重Fisher误差的严格最优性，需要解决的问题是：

$min_{rank(T)≤r} ||L_B^T · (W - T · H^{-1/2}) · L_A||_F^2$

**正确的解法**：

令 $`S = L_B^T · T · (H^{-1/2} · L_A)`$，由于 $`L_B, H^{-1/2}, L_A` $可逆：

$rank(S) = rank(T)$

问题等价于：

$min_{rank(S)≤r} ||L_B^T · W · L_A - S||_F^2$

最优解为：

S* = [L_B^T · W · L_A]_r

对应的 `T*` 为：

$T* = L_B^{-T} · S* · (H^{-1/2} · L_A)^{-1}$
   $= L_B^{-T} · [L_B^T · W · L_A]_r · L_A^{-1} · H^{1/2}$

**关键观察**：

这个最优解 `T*` 与 `G(β)` 和 `β` 都无关！它直接对 $`L_B^T · W · L_A` $做截断。

这说明：**如果目标是严格最小化权重Fisher误差，那么CEALC框架（引入H和Δ的处理）本身就不是最优路径。**

这也解释了为什么我们应该将定理陈述为"G的Fisher加权重构误差"而非"W的Fisher误差"——后者的最优解与CEALC框架不兼容。

## 六、证明总结

### 6.1 定理（最终版本）

**Theorem 1（Fisher-CEALC目标的最优性）**

在假设1-3下，Fisher-Guided CEALC方法产生的解：

$T* = L_B^{-T} · [L_B^T · G(β) · L_A]_r · L_A^{-1}$

是Fisher-CEALC目标函数：

$J(T) = ||L_B^T · (G(β) - T) · L_A||_F^2$

在约束 `rank(T) ≤ r` 下的全局最优解。

特别地，对于原始CEALC方法的解 $`T_0 = [G(β)]_r`$：

$J(T*) ≤ J(T_0)$
