## 创新点一：Fisher-Guided β Selection（Fisher 引导的累积误差对齐系数选择）

------

### 1. 动机（Motivation）

#### 1.1 SAES-SVD 中 ACES 的做法

SAES-SVD 提出了 CEALC 框架，其核心目标矩阵为：

$G_ℓ(β) = W_ℓ · (H_ℓ + β·Δ_ℓ) · H_ℓ^{-1/2}$



其中：

- `H_ℓ = X_ℓ · X_ℓ^T`：压缩后输入的协方差
- `Δ_ℓ = (X_ℓ^f - X_ℓ) · X_ℓ^T`：累积误差协方差
- `β_ℓ = α_ℓ / (1 + α_ℓ) ∈ [0, 1)`：控制累积误差对齐强度

ACES 通过最大化保留能量比（RER）来选择 β：

$β* = argmax_{β_ℓ ∈ [0,1]} ρ(β_ℓ)$



$ρ(β_ℓ) = Σ_{i=1}^{r_ℓ} σ_i^2(G_ℓ(β_ℓ)) / Σ_{i=1}^{min(m_ℓ, n_ℓ)} σ_i^2(G_ℓ(β_ℓ))$



#### 1.2 ACES 的不足

ACES 仅基于数据统计量 `H_ℓ` 和 `Δ_ℓ` 选择 β，存在以下问题：

**问题一：未区分参数方向的任务敏感度**

RER 衡量的是 `G_ℓ(β)` 在普通 Frobenius 范数下截断后保留的能量占比。但不同参数方向对任务 loss 的贡献差异很大——对 loss 敏感的方向和不敏感的方向被同等对待。

**问题二：β 的选择与任务无关**

ACES 的优化目标完全由激活统计量决定，没有利用任务 loss 的梯度信息。

#### 1.3 我们的思路

引入 GFWSVD 的 Fisher 信息加权，将 β 的选择从普通 Frobenius 空间提升到 Fisher 度量空间，使截断优先保留对任务 loss 敏感的参数方向。

------

### 2. 方法详述

#### 2.1 逐层 Fisher 信息估计与 Kronecker 分解

对第 ℓ 层，计算经验 Fisher 信息矩阵：

$I_F^{(ℓ)} = (1/|D|) · Σ_{i=1}^{|D|} g_i · g_i^T$



其中 `g_i = vec(G_i)`，`G_i = ∇_{W_ℓ} L(batch_i)`。

对 `I_F^{(ℓ)}` 做 Kronecker 分解（GFWSVD Algorithm 1）：

$I_F^{(ℓ)} ≈ A_ℓ ⊗ B_ℓ$



其中 `A_ℓ ∈ R^{n×n}`（列方向），`B_ℓ ∈ R^{m×m}`（行方向）。



#### 2.2 Cholesky 分解

对 Kronecker 因子做 Cholesky 分解：

$A_ℓ = L_A^{(ℓ)} · L_A^{(ℓ)T}$
$B_ℓ = L_B^{(ℓ)} · L_B^{(ℓ)T}$

注：需要对 `A_ℓ` 和 `B_ℓ` 做阻尼正则化 `A_ℓ ← A_ℓ + λ_A·I` 以保证正定性。



#### 2.3 Fisher 空间中的目标矩阵

定义 Fisher 变换后的目标矩阵：

$G_ℓ^F(β) = L_B^{(ℓ)T} · G_ℓ(β) · L_A^{(ℓ)}$



展开为：

$G_ℓ^F(β) = L_B^{(ℓ)T} · W_ℓ · (H_ℓ + β·Δ_ℓ) · H_ℓ^{-1/2} · L_A^{(ℓ)}$



#### 2.4 在 Fisher 空间中优化 β

定义 Fisher 空间下的保留能量比：

$β* = argmax_{β_ℓ ∈ [0,1]} ρ^F(β_ℓ)$



$ρ^F(β_ℓ) = Σ_{i=1}^{r_ℓ} σ_i^2(G_ℓ^F(β_ℓ)) / Σ_{i=1}^{min(m_ℓ, n_ℓ)} σ_i^2(G_ℓ^F(β_ℓ))$



**与原 ACES 的对比**：

|              | 原 ACES              | Fisher-Guided（ours）      |
| ------------ | -------------------- | -------------------------- |
| 优化对象     | `ρ(β)` 基于 `G_ℓ(β)` | `ρ^F(β)` 基于 `G_ℓ^F(β)`   |
| 度量空间     | 普通 Frobenius 范数  | Fisher 加权 Frobenius 范数 |
| 是否感知任务 | 否                   | 是（通过梯度信息）         |

**求解方式**：沿用 SAES-SVD 的 FS-FOA（Fixed Subspace First-Order Approximation）方法，只需将其中的 `S_ℓ` 和 `D_ℓ` 替换为 Fisher 变换后的版本：

$S_ℓ^F = L_B^T · W_ℓ · H_ℓ · H_ℓ^{-1/2} · L_A = L_B^T · S_ℓ · L_A$



$D_ℓ^F = L_B^T · W_ℓ · Δ_ℓ · H_ℓ^{-1/2} · L_A = L_B^T · D_ℓ · L_A$



$G_ℓ^F(β) = S_ℓ^F + β · D_ℓ^F$

然后套用 SAES-SVD Equation 17-19 的二次方程求解流程，将 `a, b, c, A, B, C` 替换为 Fisher 变换后的内积即可。

#### 2.5 在 Fisher 空间做截断 SVD 并映回原空间

用最优 `β*` 构造 Fisher 空间目标矩阵并做 rank-r1 截断：

$[G_ℓ^F(β*)]_{r1} = Û_{r1} · Ŝ_{r1} · V̂_{r1}^T$.



映回 CEALC 的 G 空间：

$T_ℓ* = L_B^{-T} · [G_ℓ^F(β*)]_{r1} · L_A^{-1}$



映回权重空间（乘以 `H_ℓ^{-1/2}`）：

$W_ℓ^{(1)} = T_ℓ* · H_ℓ^{-1/2}$

​     $= L_B^{-T} · Û_{r1} · Ŝ_{r1} · V̂_{r1}^T · L_A^{-1} · H_ℓ^{-1/2}$



对应的低秩因子分解：

$A_ℓ^{(1)} = L_B^{-T} · Û_{r1} · Ŝ_{r1}^{1/2}$
$B_ℓ^{(1)} = Ŝ_{r1}^{1/2} · V̂_{r1}^T · L_A^{-1} · H_ℓ^{-1/2}$



验证：

$A_ℓ^{(1)} · B_ℓ^{(1)} = L_B^{-T} · Û_{r1} · Ŝ_{r1} · V̂_{r1}^T · L_A^{-1} · H_ℓ^{-1/2} = W_ℓ^{(1)}  ✓$

### 3. 理论分析

#### 3.1 预备引理

**Lemma 1（Eckart-Young-Mirsky 定理）**

对任意矩阵 `M`，截断 SVD `[M]_r` 是 `||M - N||_F^2` 在 `rank(N) ≤ r` 约束下的全局最优解。

**Lemma 2（可逆矩阵保秩）**

设 `P, Q` 可逆，则 `rank(P·M·Q) = rank(M)`。

**Lemma 3（变量替换等价性）**

设 `P, Q` 可逆。则优化问题

$min_{rank(T) ≤ r} ||P·(G - T)·Q||_F^2$



等价于

$min_{rank(S) ≤ r} ||P·G·Q - S||_F^2$



其中 `S = P·T·Q`。（由 Lemma 2 保证可行域双射。）



#### 3.2 Theorem 1：Fisher-CEALC 截断最优性

**假设**：

1. `H_ℓ = X_ℓ·X_ℓ^T + λ_H·I ≻ 0`（阻尼正定）
2. Kronecker 因子 `A_ℓ + λ_A·I ≻ 0`，`B_ℓ + λ_B·I ≻ 0`，使得 `L_A, L_B` 可逆
3. `G_ℓ(β)` 按 SAES-SVD 定义

**定理陈述**：

> 定义 Fisher-CEALC 目标函数：
>
> $J(T) = ||L_B^T · (G_ℓ(β) - T) · L_A||_F^2, s.t. rank(T) ≤ r$
>
> 则：
>
> **(1)** Fisher-Guided 方法的解
>
> $T* = L_B^{-T} · [L_B^T · G_ℓ(β) · L_A]_r · L_A^{-1}$
>
> 是 `J(T)` 的全局最优解。
>
> **(2)** 对于原始 CEALC 方法的解 `T_0 = [G_ℓ(β)]_r`，有：
>
> $J(T*) ≤ J(T_0)$

**证明**：

**Step 1：变量替换**

令 `S = L_B^T · T · L_A`（由 `L_B, L_A` 可逆，此为双射）。由 Lemma 3，原问题等价于：

$min_{rank(S) ≤ r} ||L_B^T · G_ℓ(β) · L_A - S||_F^2$



**Step 2：定义辅助矩阵**

$N = L_B^T · G_ℓ(β) · L_A$



等价问题变为：

$min_{rank(S) ≤ r} ||N - S||_F^2$



**Step 3：应用 Eckart-Young-Mirsky 定理**

由 Lemma 1，全局最优解为：

$S* = [N]_r = [L_B^T · G_ℓ(β) · L_A]_r$



最优目标值：

$||N - S*||_F^2 = Σ_{i=r+1}^{min(m,n)} σ_i^2(N)$



**Step 4：映回原变量**

$T* = L_B^{-T} · S* · L_A^{-1} = L_B^{-T} · [L_B^T · G_ℓ(β) · L_A]_r · L_A^{-1}$



由 Lemma 2：`rank(T*) = rank(S*) ≤ r`，满足秩约束。

**结论(1)证毕。** ■



**Step 5：证明结论(2)**

原始 CEALC 的解 `T_0 = [G_ℓ(β)]_r`，满足 `rank(T_0) ≤ r`，是 `J(T)` 的一个可行解。

`T*` 是 `J(T)` 的全局最优解，因此对任意可行解（包括 `T_0`）：

$J(T*) ≤ J(T_0)$

**结论(2)证毕。** ■



**展开不等式的显式形式**：

$||L_B^T · (G_ℓ(β) - T*) · L_A||_F^2 ≤ ||L_B^T · (G_ℓ(β) - [G_ℓ(β)]_r) · L_A||_F^2$



左边 = Fisher 空间截断后的残差；右边 = 普通空间截断后再投影到 Fisher 空间的残差。

#### 3.3 定理的意义说明

**Theorem 1 证明了什么**：

在 Fisher 加权 Frobenius 范数 `||L_B^T·(·)·L_A||_F` 意义下，`T*` 是目标矩阵 `G_ℓ(β)` 的最优 rank-r 近似，且不劣于原始 CEALC 的解 `T_0`。

**物理意义**：

Fisher 加权范数 `||L_B^T·(G - T)·L_A||_F^2` 对应的是 GFWSVD 中 loss 增量的二阶近似：

$ΔL ≈ (1/2) · vec(W - W')^T · I_F · vec(W - W')$

​    $= (1/2) · ||L_B^T · (W - W') · L_A||_F^2$

因此 Theorem 1 保证：**Fisher-Guided 截断在 loss 增量意义下不劣于原始 CEALC 截断**——即压缩后模型 loss 的增量更小（或相同）。

**Remark**：Theorem 1 证明的是 G 空间的最优性（`G_ℓ(β)` 到 `T` 的距离），而非直接证明 W 空间的最优性。但由于 $W_ℓ^{(1)} = T·H_ℓ^{-1/2}$ 是线性映射，G 空间的改进会传导到权重空间的输出质量上。特别地，当 `L_A` 与 `H_ℓ^{-1/2}` 可交换时，G 空间的最优性严格等价于 W 空间的最优性。

### 4. 整体总结

| 公式/符号                               | 含义                         | 来源                  |
| --------------------------------------- | ---------------------------- | --------------------- |
| $G_ℓ(β) = W_ℓ·(H_ℓ + β·Δ_ℓ)·H_ℓ^{-1/2}$ | CEALC 目标矩阵               | SAES-SVD Eq.10        |
| $G_ℓ^F(β) = L_B^T·G_ℓ(β)·L_A$           | Fisher 变换后的目标矩阵      | 本文创新              |
| $ρ^F(β)$                                | Fisher 空间下的保留能量比    | 本文创新（改进 ACES） |
| $T* = L_B^{-T}·[G_ℓ^F(β*)]_r·L_A^{-1}$  | Fisher-Guided 截断的解       | 本文创新              |
| $J(T*) ≤ J(T_0)$                        | 核心理论保证                 | Theorem 1             |
| $I_F ≈ A⊗B$                             | Fisher 信息的 Kronecker 近似 | GFWSVD                |
| $A = L_A·L_A^T, B = L_B·L_B^T$          | Cholesky 分解                | GFWSVD                |

**创新点一的一句话总结**：将 SAES-SVD 的 β 选择和 SVD 截断从普通 Frobenius 空间提升到 Fisher 度量空间，通过任务梯度信息引导压缩优先保留对 loss 敏感的方向，理论上保证不劣于原始 CEALC（Theorem 1）。
