## Lemma 1：两阶段压缩优越性

- `X_ℓ`：压缩后传播的激活（来自已压缩的上游层）

$G_ℓ(β*) ≈ Ũ_{r1} · Σ_{r1} · Ṽ_{r1}^T$

**Stage 2（特征空间残差补偿）**：

对 `E_ℓ'` 做 rank-r2 截断SVD，映回原空间得到补偿矩阵 `W_ℓ^{comp}`。

最终：

$W_ℓ^{(2)} = W_ℓ^{(1)} + W_ℓ^{comp}$

### 三、预备引理

**Lemma A（Eckart-Young-Mirsky 定理）**

对任意矩阵 `M`，其最优 rank-k 近似为 `[M]_k`（截断SVD），且：

$||M - [M]_k||_F^2 = Σ_{i=k+1}^{min(m,n)} σ_i^2(M)$



**Lemma B（EoRA 特征空间等价性，来自 EoRA Theorem 1）**

设 $X · X^T = Q · Λ · Q^T$，$Q' = Q · Λ^{1/2}$。对任意矩阵 `Δ` 和低秩矩阵 `BA`：

$||ΔX - BAX||_F^2 = ||ΔQ' - BAQ'||_F^2 = ||Δ' - B'A'||_F^2$

其中 `Δ' = Δ · Q'`，`B'A' = BA · Q'`。

因此，在特征空间对 `Δ'` 做 rank-r 截断SVD后映回原空间，等价于最小化 `||ΔX - BAX||_F^2`。



**Lemma C（奇异值穿插不等式，来自 ResSVD Lemma 3.2）**

设 `M_r` 是 `M` 的最优 rank-r 近似，残差 `R = M - M_r`，则：

$σ_i(R) ≤ σ_{r+i}(M), ∀ 1 ≤ i ≤ min(m,n) - r$

### 四、严格证明

**Step 1：定义误差矩阵**

定义 Stage 1 的权重误差：

$ΔW = W_ℓ - W_ℓ^{(1)}$



两阶段方法的输出误差为：

$||(W_ℓ - W_ℓ^{(2)}) · X_ℓ||_F^2 = ||(ΔW - W_ℓ^{comp}) · X_ℓ||_F^2$



**Step 2：应用 EoRA 特征空间等价性（Lemma B）**

由 Lemma B，Stage 2 在特征空间对 $`E_ℓ' = ΔW · Q' = (W_ℓ - W_ℓ^{(1)}) · Q · Λ^{1/2}` $做 rank-r2 截断SVD，等价于求解：

$min_{rank(C) ≤ r2} ||ΔW · X_ℓ - C · X_ℓ||_F^2$



记最优解为 `W_ℓ^{comp}`，即 Stage 2 的补偿矩阵。

由 EoRA 的等价性，最优 rank-r2 补偿的输出误差为：

$||(ΔW - W_ℓ^{comp}) · X_ℓ||_F^2 = ||E_ℓ' - [E_ℓ']_{r2}||_F^2$



其中 `[E_ℓ']_{r2}` 是 `E_ℓ'` 的 rank-r2 截断SVD。

由 Lemma A：

$||E_ℓ' - [E_ℓ']_{r2}||_F^2 = Σ_{i=r2+1}^{min(m,n)} σ_i^2(E_ℓ')     ... (*)$



**Step 3：分析单阶段方法的输出误差**

单阶段方法：直接对 `G_ℓ(β*)` 做 rank-r 截断，得到 `W̃_ℓ = [G_ℓ(β*)]_r · H_ℓ^{-1/2}`。

由 SAES-SVD 的等价变换（Theorem 4.1），CEALC 目标 `||A·B·X_ℓ - Z_ℓ||_F^2` 等价于 `||A·B·H^{1/2} - G(β)||_F^2`。

对于当前输入下的输出误差，我们关注：

$||(W_ℓ - W̃_ℓ) · X_ℓ||_F^2$



同样利用 Lemma B 的特征空间等价性，定义 `F_ℓ' = (W_ℓ - W̃_ℓ) · Q · Λ^{1/2}`，则：

$||(W_ℓ - W̃_ℓ) · X_ℓ||_F^2 = ||F_ℓ'||_F^2 = Σ_{i=1}^{min(m,n)} σ_i^2(F_ℓ')     ... (**)$



**Step 4：建立两者之间的关系**

现在需要比较 `(*)` 和 `(**)`。

关键观察：`E_ℓ'` 和 `F_ℓ'` 都是 `W_ℓ` 近似误差在特征空间中的表示，区别在于：

- `E_ℓ' = (W_ℓ - W_ℓ^{(1)}) · Q'`：Stage 1 用了 rank-r1 → 残差有 rank-r2 补偿机会
- `F_ℓ'`：单阶段 rank-r → 无额外补偿

这里需要换一个角度比较。**直接比较输出误差本身。**

两阶段方法最终的压缩权重为 `W_ℓ^{(2)} = W_ℓ^{(1)} + W_ℓ^{comp}`，其中：

- `W_ℓ^{(1)}`：rank ≤ r1
- `W_ℓ^{comp}`：rank ≤ r2

因此 `W_ℓ^{(2)}` 的 rank ≤ r1 + r2 = r。

而 `W̃_ℓ` 的 rank ≤ r。

两者都是对 `W_ℓ` 的 rank 不超过 r 的近似。



**Step 5：证明两阶段方法不劣于单阶段**

定义目标函数：

$L(W') = ||(W_ℓ - W') · X_ℓ||_F^2$



我们证明 `L(W_ℓ^{(2)}) ≤ L(W̃_ℓ)`。

由 Lemma B，`L(W')` 等价于在特征空间最小化 `||(W_ℓ - W') · Q'||_F^2`。

定义 `Φ(W') = ||(W_ℓ - W') · Q'||_F^2`，则 `L(W') = Φ(W')`。

单阶段方法做的是：

$W̃_ℓ = argmin_{rank(W') ≤ r} ||(W_ℓ · Q') - (W' · Q')||_F^2$



由 Lemma A，最优解给出：

$Φ(W̃_ℓ) = Σ_{i=r+1}^{min(m,n)} σ_i^2(W_ℓ · Q')     ... (I)$



两阶段方法做的是：

Stage 1：

```
W_ℓ^{(1)} · Q' = 对 W_ℓ · Q' 的 rank-r1 近似（不一定是最优的，因为 Stage 1 是在 G(β*) 空间做的，不是直接在 W·Q' 空间）
```

但是 Stage 2 **是在正确的空间做最优补偿的**。Stage 2 解的是：

$min_{rank(C) ≤ r2} ||(W_ℓ - W_ℓ^{(1)} - C) · X_ℓ||_F^2$



等价于对 `E_ℓ' = (W_ℓ - W_ℓ^{(1)}) · Q'` 做 rank-r2 截断SVD。

因此：

$Φ(W_ℓ^{(2)}) = ||(W_ℓ - W_ℓ^{(1)} - W_ℓ^{comp}) · Q'||_F^2
             = ||E_ℓ' - [E_ℓ']_{r2}||_F^2
             = Σ_{i=r2+1}^{min(m,n)} σ_i^2(E_ℓ')     ... (II)$



**Step 6：核心不等式**

由 Lemma C（奇异值穿插不等式）：

`E_ℓ'` 可以看作 `W_ℓ · Q'` 减去一个 rank-r1 矩阵 `W_ℓ^{(1)} · Q'` 的残差。

注意：`W_ℓ^{(1)} · Q'` 不一定是 `W_ℓ · Q'` 的**最优** rank-r1 近似（因为 Stage 1 是在 `G(β*)` 空间做的截断，不是在 `W·Q'` 空间），所以不能直接应用 Lemma C。

**但这没关系。** 我们可以这样论证：

设 `W_ℓ · Q'` 的奇异值为 `σ_1 ≥ σ_2 ≥ ... ≥ σ_p`（`p = min(m,n)`）。

由 (I)：

$Φ(W̃_ℓ) = Σ_{i=r+1}^{p} σ_i^2(W_ℓ · Q') = σ_{r+1}^2 + σ_{r+2}^2 + ... + σ_p^2$



由 (II)：

$Φ(W_ℓ^{(2)}) = Σ_{i=r2+1}^{p} σ_i^2(E_ℓ')$



由矩阵扰动的 Weyl 不等式，对于 `E_ℓ' = W_ℓ · Q' - W_ℓ^{(1)} · Q'`，其中 `rank(W_ℓ^{(1)} · Q') ≤ r1`：

$σ_i(E_ℓ') ≤ σ_i(W_ℓ · Q') , ∀ i$



这是因为减去一个矩阵后，奇异值不会增加（这不一般成立）。

**修正：** 上面这步不对。减去任意矩阵后奇异值可以增大。我们需要用更精确的工具。



**Step 6（修正）：直接用可行性论证**

回到最本质的论证：

两阶段方法给出的 `W_ℓ^{(2)}` 满足 `rank(W_ℓ^{(2)}) ≤ r1 + r2 = r`。

单阶段最优解 `W̃_ℓ` 是在 **同一个可行域** `{W' : rank(W') ≤ r}` 中，使 `||(W_ℓ - W') · X_ℓ||_F^2` 最小的解。

因此：

$||(W_ℓ - W̃_ℓ) · X_ℓ||_F^2 ≤ ||(W_ℓ - W_ℓ^{(2)}) · X_ℓ||_F^2$



**等等，这恰好是反过来的不等式！** 单阶段在输出误差意义下做全局最优 rank-r 近似，一定不会差于两阶段。

------

### 五、重新审视引理的正确陈述

上面的推导揭示了一个关键事实：**如果单阶段和两阶段都在同一个目标函数** $`||(W - W') · X||_F^2`$ **下比较，单阶段的全局 rank-r 最优解一定不劣于两阶段。**

这意味着你文件中 Lemma 1 的不等式方向需要修正。两阶段的优势不在于同一目标下的误差更小，而在于：

**两阶段方法优化的是 SAES-SVD 的复合目标（含累积误差对齐项），而非单纯的** `||(W-W')·X||_F^2`。

------

### 六、修正后的 Lemma 1

> **Lemma 1（两阶段压缩的补偿优越性）**
>
> **假设**：
>
> 1. `H_ℓ ≻ 0`
> 2. Stage 1 对 `G_ℓ(β*)` 做 rank-r1 截断，得到 `W_ℓ^{(1)}`
> 3. `W_ℓ^{one}` 为对 `G_ℓ(β*)` 做 rank-r（= r1 + r2）截断的单阶段结果
> 4. Stage 2 对残差 $`E_ℓ' = (W_ℓ - W_ℓ^{(1)}) · Q · Λ^{1/2}`$ 做 rank-r2 最优截断
>
> **结论**：两阶段方法在 **当前输入下的输出误差** 满足：
>
> $||(W_ℓ - W_ℓ^{(2)}) · X_ℓ||_F^2 ≤ ||(W_ℓ - W_ℓ^{(1)}) · X_ℓ||_F^2$
>
> 即：Stage 2 的残差补偿**严格减少**（或不增加）Stage 1 的输出误差。

### 七、修正后的证明

**证明**：

Stage 1 的输出误差为：

$L_1 = ||(W_ℓ - W_ℓ^{(1)}) · X_ℓ||_F^2$



Stage 2 求解：

$min_{rank(C) ≤ r2} ||(W_ℓ - W_ℓ^{(1)} - C) · X_ℓ||_F^2$



令 $`ΔW = W_ℓ - W_ℓ^{(1)}`$。由 Lemma B（EoRA 特征空间等价性）：

Code

$||（ΔW - C）· X_ℓ||_F^2 = ||(ΔW - C) · Q · Λ^{1/2}||_F^2 = ||E_ℓ' - C · Q · Λ^{1/2}||_F^2$



最优 rank-r2 解为 `W_ℓ^{comp}`，使得 `W_ℓ^{comp} · Q' = [E_ℓ']_{r2}`。

因此：

$L_2 = ||(W_ℓ - W_ℓ^{(2)}) · X_ℓ||_F^2
    = ||(ΔW - W_ℓ^{comp}) · X_ℓ||_F^2
    = ||E_ℓ' - [E_ℓ']_{r2}||_F^2
    = Σ_{i=r2+1}^{p} σ_i^2(E_ℓ')$



而 `C = 0`（不做任何补偿）也是一个可行解（rank = 0 ≤ r2），此时误差为：

$||(ΔW - 0) · X_ℓ||_F^2 = ||E_ℓ'||_F^2 = Σ_{i=1}^{p} σ_i^2(E_ℓ') = L_1$



由于 `W_ℓ^{comp}` 是最优解：

$L_2 = Σ_{i=r2+1}^{p} σ_i^2(E_ℓ') ≤ Σ_{i=1}^{p} σ_i^2(E_ℓ') = L_1$



即：

$||(W_ℓ - W_ℓ^{(2)}) · X_ℓ||_F^2 ≤ ||(W_ℓ - W_ℓ^{(1)}) · X_ℓ||_F^2$



**证毕。** ■

# 二、纠正

## 一、创新点二需要证明的核心结论

你的 Stage 2 的目的是：在 Stage 1 压缩之后，用额外的 rank-r2 预算去**补偿输出误差**。那么你需要证明的是：

> **加了 Stage 2 之后，输出误差严格不大于只做 Stage 1 的误差。**

即：

$||(W_ℓ - W_ℓ') · X_ℓ||_F^2 ≤ ||(W_ℓ - W_ℓ^{(1)}) · X_ℓ||_F^2$



但这里有一个微妙之处：你的 `E_ℓ` 定义是 `W_ℓ · X_ℓ^f - W_ℓ^{(1)} · X_ℓ`，涉及**两种不同的输入**（`X_ℓ^f` 和 `X_ℓ`），所以你实际要证明的是**补偿在混合误差意义下的有效性**。

## 二、当前流程的关键问题

### 问题1：Stage 2 的补偿目标和最终评价指标不匹配

你 Stage 2 做的事情是：

- 定义误差 `E_ℓ = W_ℓ · X_ℓ^f - W_ℓ^{(1)} · X_ℓ`（混合输入）
- 通过最小二乘回归得到 `ΔW_ℓ^{LS} = E_ℓ · X_ℓ^T · H_ℓ^{-1}`
- 这个 `ΔW_ℓ^{LS}` 最小化的是 `||ΔW · X_ℓ - E_ℓ||_F^2`

展开这个目标：

$||ΔW · X_ℓ - E_ℓ||_F^2 = ||ΔW · X_ℓ - (W_ℓ · X_ℓ^f - W_ℓ^{(1)} · X_ℓ)||_F^2$



当 `ΔW = ΔW_^{LS}` 时，最终前向传播为：

$W_ℓ' · X_ℓ = (W_ℓ^{(1)} + W_ℓ^{comp}) · X_ℓ$



你希望 `W_ℓ' · X_ℓ ≈ W_ℓ · X_ℓ^f`，即：

$W_ℓ^{comp} · X_ℓ ≈ E_ℓ = W_ℓ · X_ℓ^f - W_ℓ^{(1)} · X_ℓ$



这就是说 Stage 2 在最小化：

$||W_ℓ^{comp} · X_ℓ - E_ℓ||_F^2$



**这个目标是合理的**，但你需要明确：最终评价指标到底是什么？

## 四、针对你当前流程的修正建议

你的 Step 2（最小二乘回归）其实是多余的。更直接的做法是模仿 EoRA：

**简化的 Stage 2**：

Step 1：定义输出误差

$E_ℓ = W_ℓ · X_ℓ^f - W_ℓ^{(1)} · X_ℓ$



Step 2：直接在特征空间求解 rank-r2 最优补偿

目标：

$min_{rank(C) ≤ r2} ||C · X_ℓ - E_ℓ||_F^2$



由 EoRA 的 Theorem 1，对 `X_ℓ · X_ℓ^T = Q · Λ · Q^T`，令 `Q' = Q · Λ^{1/2}`：

$||C · X_ℓ - E_ℓ||_F^2 等价于最小化 ||C · Q' - E_ℓ · Q'||_F^2（当 Λ 正定时不严格等价，需要条件）$



实际上正确的等价变换是（由 EoRA 证明）：

$min_{rank(C) ≤ r2} ||C · X_ℓ - E_ℓ||_F^2$



对 `E_ℓ · Q' = E_ℓ · Q · Λ^{1/2}` 做 rank-r2 截断SVD，得到 `[E_ℓ · Q']_{r2}`，然后映回：

$W_ℓ^{comp} = [E_ℓ · Q']_{r2} · Λ^{-1/2} · Q^T$



Step 3：最终权重

$W_ℓ' = W_ℓ^{(1)} + W_ℓ^{comp}$



**好处**：省去了最小二乘回归步骤，直接对输出误差 `E_ℓ` 在特征空间做低秩近似，理论上更干净。

## 五、如果你坚持保留最小二乘步骤

# 三、重新论证

## 一、ResSVD 证明的本质

ResSVD 证明的是**权重空间**的不等式：

$L_C = ||W - Ŵ_r||_F^2 ≤ ||W - W_r||_F^2 = L_D$



其中 `Ŵ_r = W_{r1} + R_{r2}`（两阶段），`W_r`（单阶段 rank-r 直接截断）。

关键：ResSVD 的两阶段都是在**同一个对象 W** 上做的——Stage 1 对 W 做 rank-r1 截断，Stage 2 对残差 `R = W - W_{r1}` 再做 rank-r2 截断。所以能用 Lemma 3.2（奇异值穿插不等式）串联起来。

## 二、你的情况与 ResSVD 的本质区别

你的框架有两个不同点：

**区别1**：你的 Stage 1 不是对 W 直接做 SVD，而是对 `G(β*) = W·(H + β*Δ)·H^{-1/2}` 在 Fisher 空间做截断。所以 Stage 1 的残差 **不是** `W - W_{r1}` 这种简单形式。

**区别2**：你的误差定义涉及激活：`E_ℓ = W·X^f - W^{(1)}·X`，目标是最小化**输出误差**而非权重误差。

所以你**不能直接套用 ResSVD 的证明框架**。

## 三、你应该证什么

SVD 系列论文的优化目标主要两类：

| 类型                 | 目标               | 代表论文                |
| -------------------- | ------------------ | ----------------------- |
| 权重误差             | `||W - W'||_F^2`   | ResSVD                  |
| 激活误差（输出误差） | `||WX - W'X||_F^2` | SVD-LLM, SAES-SVD, EoRA |

你的框架基于 SAES-SVD 改进，SAES-SVD 本身就是优化激活误差的，所以你的证明目标应该落在**激活误差**上。

### 我建议你证明的结论

> **Lemma 1（两阶段压缩在激活误差下的有效性）**
>
> 设 Stage 1 得到 `W_ℓ^{(1)}`（rank ≤ r1），定义输出误差 $`E_ℓ = W_ℓ·X_ℓ^f - W_ℓ^{(1)}·X_ℓ`$。Stage 2 对 `E_ℓ` 在特征空间做 rank-r2 最优近似，得到补偿 `W_ℓ^{comp}`。
>
> 则：
>
> $||W_ℓ·X_ℓ^f - (W_ℓ^{(1)} + W_ℓ^{comp})·X_ℓ||_F^2 ≤ ||W_ℓ·X_ℓ^f - W_ℓ^{(1)}·X_ℓ||_F^2$
>
> 
>
> 即：
>
> $||E_ℓ - W_ℓ^{comp}·X_ℓ||_F^2 ≤ ||E_ℓ||_F^2$
>
> 

**这个不等式的物理意义**：加了 Stage 2 残差补偿后，压缩模型的输出与全精度参考输出之间的差距**严格不增加**。

## 四、这个结论是否"够分量"

你可能觉得这个太简单了。确实，单独看这个不等式比较弱。但你要注意，你的论文有**两个创新点组合**：

| 创新点                     | 理论保证                         |
| -------------------------- | -------------------------------- |
| 创新点1（Fisher-Guided β） | Theorem 1：Fisher 空间截断最优性 |
| 创新点2（残差补偿）        | Lemma 1：补偿有效性              |

两者组合的故事是：**Stage 1 通过 Fisher 信息选出对 loss 最敏感的方向来压缩，Stage 2 通过特征空间残差补偿进一步减少剩余误差**。Lemma 1 保证 Stage 2 不会"帮倒忙"。

如果你想让 Lemma 1 更强，可以加一个**推论**，仿照 ResSVD 的形式但在激活空间：

> **Corollary 1（特征空间补偿优于朴素补偿）**
>
> 设 `W_ℓ^{naive}` 为对权重残差 `W_ℓ - W_ℓ^{(1)}` 直接做 rank-r2 截断SVD的结果（不经过特征空间变换）。则：
>
> Code
>
> $||E_ℓ - W_ℓ^{comp}·X_ℓ||_F^2 ≤ ||E_ℓ - W_ℓ^{naive}·X_ℓ||_F^2$
>
> 

这个推论说的是：**在特征空间做补偿比在权重空间直接做补偿更好**，这正是 EoRA 的核心贡献，也是你把 EoRA 思想引入 SAES-SVD 框架的理论依据。

## 五、证明应该怎么走

### Lemma 1 证明（简洁版）

$min_{rank(C) ≤ r2} ||E_ℓ - C·X_ℓ||_F^2$

`W_ℓ^{comp}` 是该问题的最优解（通过 EoRA 特征空间方法求得），因此：

Code

$||E_ℓ - W_ℓ^{comp}·X_ℓ||_F^2 ≤ ||E_ℓ - 0·X_ℓ||_F^2 = ||E_ℓ||_F^2$

证毕。



### Corollary 1 证明（核心）

这个才是你真正需要花篇幅写的。证明思路：

由 EoRA Theorem 1，特征空间截断等价于直接最小化 $`||E_ℓ - C·X_ℓ||_F^2`$。而朴素SVD截断最小化的是$ `||ΔW - C||_F^2`$（权重误差），后者不等价于前者（除非 `X_ℓ·X_ℓ^T = I`）。

# 四、创新点2的E_ℓ误差定义解释

## 一、你的 E_ℓ 并不是"自创"的，它有明确的来源

你的 `E_ℓ = W_ℓ·X_ℓ^f - W_ℓ^{(1)}·X_ℓ` 拆开看：

Code

$E_ℓ = W_ℓ·X_ℓ^f - W_ℓ^{(1)}·X_ℓ$
    $= (W_ℓ·X_ℓ^f - W_ℓ·X_ℓ) + (W_ℓ·X_ℓ - W_ℓ^{(1)}·X_ℓ)$
    $= W_ℓ·(X_ℓ^f - X_ℓ) + (W_ℓ - W_ℓ^{(1)})·X_ℓ$

- 第一项 `W_ℓ·(X_ℓ^f - X_ℓ)`：**上游累积误差**造成的输出偏差
- 第二项 `(W_ℓ - W_ℓ^{(1)})·X_ℓ`：**本层压缩误差**造成的输出偏差

**这正是 SAES-SVD 的核心思想**——CEALC 的目标函数（Equation 3）同时优化层内重构误差和累积误差对齐。你的 `E_ℓ` 就是这两项误差的总和，是 SAES-SVD 框架的自然推论，不是凭空定义的。

## 二、与其他论文优化目标的关系

| 论文     | 优化目标                     | 你的 E_ℓ 的对应关系                                          |
| -------- | ---------------------------- | ------------------------------------------------------------ |
| SVD-LLM  | `||WX - W'X||_F^2`           | 当 `X_ℓ^f = X_ℓ`（无累积误差）时，`E_ℓ` 退化为 `(W-W')·X`，与 SVD-LLM 一致 |
| ResSVD   | `||W - W'||_F^2`             | 当 `X = I`（单位阵）时，激活误差退化为权重误差               |
| SAES-SVD | Eq.3 同时含层内误差+累积对齐 | `E_ℓ` 正好是 SAES-SVD 两项目标的叠加                         |
| EoRA     | `||ΔW·X - BA·X||_F^2`        | EoRA 补偿的也是输出误差，只是不含累积项                      |

所以你的 `E_ℓ` 是这些目标的**推广形式**，不是更弱，而是**更一般**。

## 三、论文中怎么讲这个故事

建议的叙事结构：

### 第一段：指出现有方法的不足

> 现有 SVD 残差补偿方法（如 ResSVD）仅在权重空间最小化 `||W - W'||_F^2`，忽略了两个因素：（1）输入激活的分布不均匀性——不同方向的误差对输出的影响不同；（2）上游压缩层引入的累积误差——逐层压缩时，当前层的输入已经偏离了全精度输入。

### 第二段：定义你的补偿目标并说明动机

> 为解决上述问题，我们定义全局输出对齐误差：
>
> $E_ℓ = W_ℓ·X_ℓ^f - W_ℓ^{(1)}·X_ℓ$
>
> 
>
> 该误差度量的是：压缩后模型在第 ℓ 层的输出与全精度模型输出之间的总偏差，天然包含了本层截断误差和上游累积误差两部分。这与 SAES-SVD 的 CEALC 目标（Eq.3）一脉相承——Stage 1 的 CEALC 通过 β 平衡两项误差来做主压缩，Stage 2 则对剩余的总偏差做进一步补偿。

### 第三段：与标准激活误差目标的���系

> 注意到，当上游层无压缩误差（即 `X_ℓ^f = X_ℓ`）时，`E_ℓ` 退化为标准的激活误差 `(W_ℓ - W_ℓ^{(1)})·X_ℓ`，此时我们的方法与 SVD-LLM 的优化目标一致。因此，我们的框架是标准激活误差优化的推广，额外考虑了累积误差的补偿。

### 第四段：理论保证

> **Lemma 1**：Stage 2 的特征空间残差补偿保证输出对齐误差严格不增加：

> $||E_ℓ - W_ℓ^{comp}·X_ℓ||_F^2 ≤ ||E_ℓ||_F^2$
>
> **Corollary 1**：相比于在权重空间直接做朴素 SVD 补偿（如 ResSVD），特征空间补偿在激活误差意义下严格更优：
