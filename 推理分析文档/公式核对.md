## 一、SAES-SVD基线公式核对

### 1.1 CEALC目标函数（Equation 3）

$argmin_{A_ℓ, B_ℓ} ||（A_ℓ·B_ℓ - W_ℓ)·X_ℓ||_F^² + α_ℓ·||A_ℓ·B_ℓ·X_ℓ - W_ℓ·X_ℓ^f||_F^²$

**请核对**：这是否与SAES-SVD论文的Equation 3一致？

答：基本一致，但关于范数的平方表示不对，不是`_F²`，应该是`_F^²`.



### 1.2 等价变换（Equation 4-5）

Equation 4（展开后的等价形式）：

$||(A_ℓ·B_ℓ·X_ℓ - T_ℓ)||_F^² + α_ℓ·||(A_ℓ·B_ℓ·X_ℓ - R_ℓ)||_F^²\\= (1 + α_ℓ)·||A_ℓ·B_ℓ·X_ℓ - Z_ℓ||_F^² + C_ℓ$

其中：
- $T_ℓ = W_ℓ·X_ℓ$ （当前输入下的输出）
- $R_ℓ = W_ℓ·X_ℓ^f$ （全精度输入下的输出）
- $C_ℓ = ||T_ℓ||_F^² + α_ℓ·||R_ℓ||_F^² - (1+α_ℓ)·||Z_ℓ||_F^²$ （常数项）

Equation 5（加权平均）：

$Z_ℓ = (T_ℓ + α_ℓ·R_ℓ) / (1 + α_ℓ) = W_ℓ · (X_ℓ + α_ℓ·X_ℓ^f) / (1 + α_ℓ)$

**请核对**：变换过程是否正确？

答：基本正确，同样关于范数的平方表示不对，不是`_F²`，应该是`_F^²`.



### 1.3 二阶统计量表示（Equation 9）

**我理解的公式**：

$Z_ℓ·X_ℓ^T = W_ℓ·(X_ℓ·X_ℓ^T + α_ℓ·X_ℓ^f·X_ℓ^T) / (1 + α_ℓ)
          = W_ℓ·(H_ℓ + β_ℓ·Δ_ℓ)$

其中：
- $H_ℓ = X_ℓ·X_ℓ^T$ （输入协方差）
- $Δ_ℓ = (X_ℓ^f - X_ℓ)·X_ℓ^T$ （累积误差协方差）
- $β_ℓ = α_ℓ / (1 + α_ℓ) ∈ [0, 1)$

**请核对**：$Δ_ℓ$的定义是否正确？特别是：

- 是 $(X_ℓ^f - X_ℓ)·X_ℓ^T$ 还是 $X_ℓ^f·X_ℓ^T - X_ℓ·X_ℓ^T$？

答：正确， $(X_ℓ^f - X_ℓ)·X_ℓ^T$ 去掉括号后等价于 $X_ℓ^f·X_ℓ^T - X_ℓ·X_ℓ^T$.



### 1.4 Theorem 4.1（空间变换）

**我理解的定理内容**：

设 $H_ℓ = X_ℓ·X_ℓ^T ≻ 0$，$L_ℓ = H_ℓ^{-1/2}$，则：

$argmin_{rank(U_ℓ)≤r_ℓ} ||U_ℓ·X_ℓ - Z_ℓ||_F^² = argmin_{rank(V_ℓ)≤r_ℓ} ||V_ℓ - M_ℓ·L_ℓ||_F^²$

其中 $V_ℓ := U_ℓ·H_ℓ^{1/2}$



**请核对**：

- $M_ℓ$是什么？是$Z_ℓ·X_ℓ^T$吗？
- 变换关系是否正确？



答：$M_ℓ$的来源是论文附录中A.2提到的：$M = Z X^\top \in \mathbb{R}^{m \times p}$.

变换基本正确，但是有小错误，第一次是指数的表示有误`L_ℓ = H_ℓ^(-1/2)`应该是`L_ℓ = H_ℓ^{-1/2}`，第二也是范数的平方表示不对，不是`_F²`，应该是`_F^²`，第三是`U_ℓ`和`V_ℓ`范围不对，需要从`rank(U_ℓ)≤r`和`rank(V_ℓ)≤r`改为`rank(U_ℓ)≤r_ℓ`和`rank(V_ℓ)≤r_ℓ`.





### 1.5 最终优化目标（Equation 10）

**我理解的公式**：

$argmin_{A_ℓ, B_ℓ} ||A_ℓ·B_ℓ·H_ℓ^{1/2} - W_ℓ·(H_ℓ + β·Δ_ℓ)·H_ℓ^{-1/2}||_F^²$

定义目标矩阵：

$G_ℓ(β) = W_ℓ·(H_ℓ + β·Δ_ℓ)·H_ℓ^{-1/2}$

**请核对**：这是否是CEALC的最终形式？

答：这是CEALC的最终形式，只有一处前面出现过的小错误，`H_ℓ^(-1/2)`应该是` H_ℓ^{-1/2}`.





### 1.6 闭式解（Equation 11-12）

**我理解的公式**：

Equation 11（截断SVD）：

$T_ℓ* = [G_ℓ]_r = Ũ_r·Σ_r·Ṽ_r^T$



Equation 12（低秩因子）：

$A_ℓ = Ũ_r·Σ_r^{1/2}$
$B_ℓ = Σ_r^{1/2}·Ṽ_r^T·L_ℓ$

其中 $L_ℓ = H_ℓ^{-1/2}$



**请核对**：闭式解形式是否正确？

答：闭式解形式有小错误，式中的`r`应该是`r_ℓ`，

公式11应为：$T_ℓ* = [G_ℓ]_{r_ℓ} = Ũ_{r_ℓ}·Σ_{r_ℓ}·Ṽ_{r_ℓ}^T$.

公式12应为：

$A_ℓ = Ũ_{r_ℓ}·Σ_{r_ℓ}^{1/2}$
$B_ℓ = Σ_{r_ℓ}^{1/2}·Ṽ_{r_ℓ}^T·L_ℓ$.



### 1.7 ACES的β优化（Section 4.2）

**我理解的优化目标**：

$β* = argmax_{β} ρ(β)$

$ρ(β) = Σ_{i=1}^r σ_i²(G(β)) / Σ_{i=1}^n σ_i²(G(β))\\
     = 保留能量 / 总能量\\
     = RER (Retained Energy Ratio)$



**请核对**：

- ρ(β)的定义是否正确？
- 是最大化还是最小化？



答：优化目标和ρ(β)的正确定义应该如下：

$β* = \arg\max_{\beta_\ell \in [0,1]} ρ(β_\ell)$

$ρ(\beta_\ell) = \arg\max_{\beta_\ell \in [0,1]} \frac{ \sum_{i=1}^{r_\ell} \sigma_i^2( G_\ell(\beta_\ell) ) }{ \sum_{i=1}^{\min(m_\ell, n_\ell)} \sigma_i^2( G_\ell(\beta_\ell) ) }\\
     = 保留能量 / 总能量\\
     = RER (Retained Energy Ratio)$

### 2.3 Theorem 1（GFWSVD核心定理）

**我理解的定理内容**：

设 $W ∈ R^{n×m}$ 为权重矩阵，若满足：

1. 损失函数来自MLE问题
2. 经验Fisher信息具有Kronecker积结构 $I_F = A ⊗ B$
3. 权重W服从矩阵变量正态分布 $MVN(W^*, B^{-1}, A^{-1})$

$Ŵ_r = L_B^{-T} · W̃_r · L_A^{-1}$

其中：
- $A = L_A·L_A^T, B = L_B·L_B^T$ （Cholesky分解）
- $W̃ = L_B^T·W^*·L_A$ （辅助矩阵）
- $W̃_r = SVD_r(W̃) $（rank-r截断SVD）



**请核对**：

- 定理条件是否完整？
- Cholesky分解的方向（$A = L_A·L_A^T$ vs $A = L_A^T·L_A$）？

答：Theorem 1 定理条件基本完整，但需要注意是的$W*$应该为写为$W^*$.

Cholesky分解的方向为$A = L_A·L_A^T$.

## 三、残差补偿参考公式核对

### 3.1 ResSVD两阶段框架

**我理解的核心思想**：

原始权重分解为两段：

$W ≈ W_i + W_r$

其中：
- $W_i = U_{r1}·Σ_{r1}·V_{r1}^T$ （初始阶段，rank = r1）
- $W_r = Ũ_{r2}·Σ̃_{r2}·Ṽ_{r2}^T$ （残差阶段，rank = r2）
- 总秩：$r = r1 + r2$



**核心不等式**：

$||W - (W_i + W_r)||_F ≤ ||W - W_{single}||_F$

其中 $W_{single} = U_r·Σ_r·V_r^T$ 是直接做rank-r的单次截断





**请核对**：

- 残差是对权重做SVD还是对输出误差做SVD？
- 不等式方向是否正确？

答：残差是对误差做SVD，$W_{r} = W - W_{i_1}$，不等式方向正确.

### 3.2 EoRA特征空间补偿

**我理解的核心公式**：

目标：最小化输出误差

$min ||ΔW · X||_F²$

其中 $ΔW = W - W'$ 是压缩误差



特征分解变换：

$X·X^T = Q·Λ·Q^T$

$ΔW'=||ΔW · Q'||_F^² = ||ΔW · Q · Λ^{(1/2)}||_F^²$ （在$Λ^{(1/2)}$意义下）



最优补偿：

对 $ΔW' = ΔW · Q · Λ^{(1/2)}$ 做SVD，得到 $ΔW'≈ Ũ·Σ̃·Ṽ^T$,定义 $B'=ŨΣ̃, A'=Ṽ^T$

逆投影回原空间：$A=A'⋅Q'^{−1}=A'⋅Λ^{-1/2}·Q^{T}$

最终前向传播：$WX+B'AX$



对于校准集的激活值 X（满足 $XX^T=QΛQ^T$）：

$||ΔWX−BAX||_F^2=trace((ΔW−BA)XX^T(ΔW−BA)^T)  =trace((ΔW−BA)QΛQ^T(ΔW−BA)^T) \\ =trace((ΔWQ−BAQ)Λ(ΔWQ−BAQ)^T)  \\ =trace((ΔWQΛ−BAQΛ)(ΔWQΛ−BAQΛ)^T) \\  =||ΔWQΛ−BAQΛ||_F^2 =||ΔW'−BAQ'||_F^2$



补偿矩阵：$W^{comp} =B'A= Ũ_r · Σ̃_r · Ṽ_r^T · Λ^{-1/2} · Q^T ​$



**请核对**：

- 变换方向（左乘Q还是右乘Q）？
- $Λ^{-1/2}$是否需要正则化处理？

答：变换方向右乘QΛ是对的，$Λ^{-1/2}​$​论文没有说需要正则化。矩阵补偿形式不对，应拆分为 B′ 和 A 两个低秩矩阵，最优补偿为：

对 $ΔW' = ΔW · Q · Λ^{(1/2)}$ 做SVD，得到 $ΔW'≈ Ũ·Σ̃·Ṽ^T$,定义 $B'=ŨΣ̃, A'=Ṽ^T$.

逆投影回原空间：$A=A'⋅Q'^{−1}=A'⋅Λ^{-1/2}·Q^{T}$.

最终前向传播：$WX+B'AX​$.

对于校准集的激活值 X（满足 $XX^T=QΛQ^T$）：

$||ΔWX−BAX||_F^2=trace((ΔW−BA)XX^T(ΔW−BA)^T)  \\ =trace((ΔW−BA)QΛQ^T(ΔW−BA)^T) \\ =trace((ΔWQ−BAQ)Λ(ΔWQ−BAQ)^T)  \\ =trace((ΔWQΛ−BAQΛ)(ΔWQΛ−BAQΛ)^T) \\  =||ΔWQΛ−BAQΛ||_F^2 =||ΔW'−BAQ'||_F^2​$

补偿矩阵：$W^{comp} =B'A= Ũ_r · Σ̃_r · Ṽ_r^T · Λ^{-1/2} · Q^T ​$.

## 五、需要你核对的关键问题清单

### 问题1：Δ_ℓ的精确定义

$Δ_ℓ = (X_ℓ^f - X_ℓ)·X_ℓ^T$   还是   $Δ_ℓ = X_ℓ^f·X_ℓ^T - X_ℓ·X_ℓ^T$ ? 

答：$Δ_ℓ$的精确定义是 $(X_ℓ^f - X_ℓ)·X_ℓ^T$ ，但去掉括号后等价于 $X_ℓ^f·X_ℓ^T - X_ℓ·X_ℓ^T$



### 问题2：CEALC闭式解的完整形式

从 $G(β) = W·(H + β·Δ)·H^{-1/2}$ 到 $W' = A·B$

A 和 B 的精确表达式是什么？

答：Equation 12（低秩因子）：$A_ℓ = Ũ_r·Σ_r^{1/2}$，$B_ℓ = Σ_r^{1/2}·Ṽ_r^T·L_ℓ$



### 问题3：GFWSVD的Cholesky分解方向

$A = L_A·L_A^T$  还是  $A = L_A^T·L_A$ ? 

答：Cholesky分解的方向为$A = L_A·L_A^T$



### 问题4：EoRA的变换方向

是对 $E·Q·Λ^{1/2}$ 做SVD，还是对 $Q^T·E·某个矩阵$ 做SVD？

答：对 $ΔW' = ΔW · Q · Λ^{(1/2)}$ 做SVD

答：补偿 $||W·X^f - W'·X||_F^²$（输出误差）
