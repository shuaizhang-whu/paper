# （一）残差补偿权重，旧版证明过程

#### Stage 1（Fisher-CEALC 主压缩）

对层 $\ell$，先用 2.1 得到 $\beta^*$，并做 Fisher-CEALC rank-$r_1$ 主压缩：

$G_\ell(\beta^*)=W_\ell\,(H_\ell+\beta^*\Delta_\ell)\,H_\ell^{-1/2}$  

$G_\ell^F(\beta^*)=L_B^{(\ell)\top}G_\ell(\beta^*)L_A^{(\ell)}$  

$T_\ell^*=L_B^{(\ell)-\top}[\,G_\ell^F(\beta^*)\,]_{r_1}L_A^{(\ell)-1}$  

$W_\ell^{(1)}=T_\ell^*H_\ell^{-1/2}$

对应低秩因子可写为：若 $[\,G_\ell^F(\beta^*)\,]_{r_1}=U\Sigma V^T$，则  

$A_\ell^{(1)}=L_B^{(\ell)-\top}U\Sigma^{1/2}$，$B_\ell^{(1)}=\Sigma^{1/2}V^TL_A^{(\ell)-1}H_\ell^{-1/2}$，且 $W_\ell^{(1)}=A_\ell^{(1)}B_\ell^{(1)}$。



#### Stage 2（特征空间残差补偿）

Step 1：计算输出误差（对齐 full-precision 参考）  

$E_\ell=W_\ell X_\ell^f - W_\ell^{(1)}X_\ell$



Step 2：将输出误差回归为“权重残差” 

先构造协方差：
$H_\ell = X_\ell X_\ell^T + \lambda_H I$

令
$\Delta W_\ell^{LS}=E_\ell X_\ell^T H_\ell^{-1}$

它是 $\min_{\Delta W}\|\Delta W X_\ell-E_\ell\|_F^2$ 的（带阻尼的）闭式解。

>我的目标是：找到一个最佳的修补权重 $\Delta W$，让它乘以输入 $X$ 后，能最大程度地去“填补”原模型和量化模型之间的输出误差。
>
>我们要解决的核心数学问题是：
>
>$$\min_{\Delta W} J(\Delta W) = \underbrace{||\Delta W \cdot X - Y||_F^2}_{\text{误差项}} + \underbrace{\lambda ||\Delta W||_F^2}_{\text{阻尼(正则)项}}$$
>
>其中 $Y =E_\ell= W_\ell X_\ell^f - W_\ell^{(1)}X_\ell$ 是我们要拟合的目标误差。
>
>下面是推导流程：
>
>------
>
>##### 第一步：拆解 Frobenius 范数
>
>Frobenius 范数的平方不好直接求导，必须用**迹（Trace）**的性质把它展开。
>
>涉及到的核心公式：
>
>1. $||A||_F^2 = \text{trace}(A A^T)$
>2. $\text{trace}(A + B) = \text{trace}(A) + \text{trace}(B)$
>
>先把损失函数 $J$ 展开：
>
>$$\begin{aligned} J &= \text{trace}\Big( (\Delta W X - Y)(\Delta W X - Y)^T \Big) + \lambda \cdot \text{trace}(\Delta W \Delta W^T) \\   &= \text{trace}\Big( (\Delta W X - Y)(X^T \Delta W^T - Y^T) \Big) + \lambda \cdot \text{trace}(\Delta W \Delta W^T) \\ \end{aligned}$$
>
>继续把括号乘开：
>
>$$\text{Inside} = \Delta W X X^T \Delta W^T - \Delta W X Y^T - Y X^T \Delta W^T + Y Y^T$$
>
>利用迹的性质 $\text{trace}(A) = \text{trace}(A^T)$，我们可以合并中间两项：
>
>$\text{trace}(\Delta W X Y^T) = \text{trace}((Y X^T \Delta W^T)^T) = \text{trace}(Y X^T \Delta W^T)$。
>
>**此时，损失函数变成了清晰的四项：**
>
>$$J = \underbrace{\text{trace}(\Delta W X X^T \Delta W^T)}_{\text{二次项}} - 2\underbrace{\text{trace}(Y X^T \Delta W^T)}_{\text{一次项}} + \underbrace{\text{trace}(Y Y^T)}_{\text{常数项}} + \lambda \underbrace{\text{trace}(\Delta W \Delta W^T)}_{\text{正则项}}$$
>
>------
>
>##### 第二步：矩阵求导
>
>现在我们要对矩阵 $\Delta W$ 求导，并令其为 0。这里需要用到两个矩阵的求导公式：
>
>- **公式 1（针对一次项）：** $\frac{\partial \text{trace}(A B)}{\partial A} = B^T$
>- 对应我们的 $\text{trace}(Y X^T \Delta W^T)$，这里 $A$ 是 $\Delta W^T$，或者直接把 $Y X^T$ 看作系数。
>- 结果是：$-2 Y X^T$
>- **公式 2（针对二次项）：** $\frac{\partial \text{trace}(A B A^T)}{\partial A} = 2AB$ （当 $B$ 对称时）
>- 对应我们的 $\text{trace}(\Delta W (X X^T) \Delta W^T)$。这里 $A=\Delta W$, $B=X X^T$。
>- 结果是：$2 \Delta W (X X^T)$
>- **公式 3（针对正则项）：** $\frac{\partial \text{trace}(A A^T)}{\partial A} = 2A$
>- 对应 $\lambda \text{trace}(\Delta W \Delta W^T)$。
>- 结果是：$2 \lambda \Delta W$
>
>**把它们合起来，导数（梯度）就是：**
>
>$$\frac{\partial J}{\partial \Delta W} = 2 \Delta W (X X^T) - 2 Y X^T + 2 \lambda \Delta W$$
>
>------
>
>##### 第三步：令梯度为 0 并求解
>
>让导数等于 0 矩阵：
>
>$$2 \Delta W (X X^T) + 2 \lambda \Delta W - 2 Y X^T = 0$$
>
>1. **约掉系数 2：**
>
>$$\Delta W (X X^T) + \lambda \Delta W = Y X^T$$
>
>2. **提取公因式 $\Delta W$：**
>
>因为 $\Delta W$ 都在左边，我们在右边提取 $(XX^T + \lambda I)$：
>
>$$\Delta W \cdot \Big( X X^T + \lambda I \Big) = Y X^T$$
>
>*(注：因为 $\lambda$ 是个标量，不能直接加在矩阵 $XX^T$ 上)*
>
>3. **右乘逆矩阵：**
>
>为了把 $\Delta W$ 分离出来，我们在等式两边同时右乘 $(X X^T + \lambda I)^{-1}$：
>
>$$\Delta W = Y X^T \cdot (X X^T + \lambda I)^{-1}$$
>
>------
>
>##### 第四步：代回变量
>
>因为 $Y = W X - \hat{W} X$ （即全精度输出 - 量化输出）。
>
>把 $Y=E_\ell$ 代回去，得到最终闭式解：
>
>$$\Delta W = (W X - \hat{W} X) X^T (X X^T + \lambda I)^{-1} = E_\ell X_\ell^T H_\ell^{-1}$$
>
>**这里的物理意义如下：**
>
>- **$Y X^T$**：是误差与输入的互相关，代表“这种误差通常伴随着什么样的输入出现”。
>- **$(X X^T + \lambda I)^{-1}$**：是输入数据的协方差矩阵的逆，用来“白化”数据，消除特征之间的相关性干扰。
>- **$\lambda I$**：防止 $X X^T$ 不可逆（也就是防止分母为 0）。
>
>推导完毕！结果类似于 Ridge Regression岭回归的闭式解标准形式。



Step 3：特征空间白化

对 $H_\ell$ 特征分解：
$H_\ell=Q_\ell\Lambda_\ell Q_\ell^T$

白化到特征空间：
$\Delta W_{\ell,\text{white}}=\Delta W_\ell^{LS}\,Q_\ell\Lambda_\ell^{1/2}$



Step 4：对 $\Delta W_{\ell,\text{white}}$ 做 rank-$r_2$ 截断 SVD  

$\Delta W_{\ell,\text{white}}\approx[\,\Delta W_{\ell,\text{white}}\,]_{r_2}=\tilde U\tilde\Sigma\tilde V^T$



Step 5：逆投影回原空间得到补偿矩阵  

$W_{\ell}^{comp}=[\,\Delta W_{\ell,\text{white}}\,]_{r_2}\,\Lambda_\ell^{-1/2}Q_\ell^T$



Step 6：最终权重  

$W_\ell' = W_\ell^{(1)} + W_{\ell}^{comp}$





# 创新点二（初次梳理）

------

### 1. 动机（Motivation）

#### 1.1 现有方法的不足

现有 SVD 系列压缩方法在残差补偿方面存在两个主要问题：

**问题一：权重空间补偿忽略激活分布**

ResSVD 等方法在权重空间最小化 `||W - W'||_F^2`，隐含假设所有方向的误差对输出的影响相同。但实际上，输入激活在不同方向上的能量分布极不均匀——某些方向的权重误差对输出影响很大，另一些几乎无影响。在权重空间做补偿无法区分这种差异。

**问题二：忽略上游累积误差**

逐层压缩时，当前层的输入 `X_ℓ` 已经偏离了全精度输入 `X_ℓ^f`。现有补偿方法（如 ResSVD、EoRA）仅考虑本层压缩误差 `(W_ℓ - W_ℓ^{(1)})·X_ℓ`，未将上游累积误差纳入补偿目标。

#### 1.2 我们的思路

在 Stage 1（Fisher-CEALC 主压缩）之后，仍然存在未被消除的输出偏差。我们提出利用额外的 rank-r2 预算，在特征空间对这一残余偏差做最优低秩补偿，从而进一步降低压缩损失。

------

### 2. 补偿目标的定义与合理性

#### 2.1 全局输出对齐误差

定义 Stage 1 压缩后的**全局输出对齐误差**：

Code

```
E_ℓ = W_ℓ · X_ℓ^f - W_ℓ^{(1)} · X_ℓ
```

将其展开：

Code

```
E_ℓ = W_ℓ·(X_ℓ^f - X_ℓ) + (W_ℓ - W_ℓ^{(1)})·X_ℓ
```

- 第一项 `W_ℓ·(X_ℓ^f - X_ℓ)`：上游累积误差造成的输出偏差
- 第二项 `(W_ℓ - W_ℓ^{(1)})·X_ℓ`：本层压缩截断造成的输出偏差

#### 2.2 与现有优化目标的关系

| 条件                        | E_ℓ 退化为                              | 对应论文      |
| --------------------------- | --------------------------------------- | ------------- |
| `X_ℓ^f = X_ℓ`（无累积误差） | `(W_ℓ - W_ℓ^{(1)})·X_ℓ`（标准激活误差） | SVD-LLM, EoRA |
| `X_ℓ = I`（单位输入）       | `W_ℓ·X_ℓ^f - W_ℓ^{(1)}`（近似权重误差） | ResSVD        |

因此 `E_ℓ` 是标准激活误差的推广形式，与 SAES-SVD 的 CEALC 目标（同时优化层内重构误差和累积误差对齐）一脉相承：Stage 1 通过 β 平衡两项误差做主压缩，Stage 2 对剩余的总偏差做进一步补偿。

#### 2.3 Stage 2 的优化目标

Stage 2 要求解的问题是：

Code

```
min_{rank(C) ≤ r2} ||E_ℓ - C · X_ℓ||_F^2
```

即：在 rank 不超过 r2 的约束下，找到补偿矩阵 `C`，使得 `C·X_ℓ` 尽可能逼近 `E_ℓ`。

补偿后，第 ℓ 层的输出变为：

Code

```
W_ℓ' · X_ℓ = (W_ℓ^{(1)} + W_ℓ^{comp}) · X_ℓ ≈ W_ℓ · X_ℓ^f
```

------

### 3. 特征空间补偿方法

#### 3.1 特征分解与白化

对输入协方差矩阵做特征分解：

Code

```
H_ℓ = X_ℓ · X_ℓ^T = Q_ℓ · Λ_ℓ · Q_ℓ^T
```

其中 `Q_ℓ` 为正交矩阵，`Λ_ℓ` 为特征值对角阵。定义白化投影矩阵：

Code

```
Q_ℓ' = Q_ℓ · Λ_ℓ^{1/2}
```

#### 3.2 投影到特征空间

将输出误差投影到特征空间：

Code

```
E_ℓ' = E_ℓ · Q_ℓ' = E_ℓ · Q_ℓ · Λ_ℓ^{1/2}
```

#### 3.3 在特征空间做 rank-r2 截断 SVD

对 `E_ℓ'` 做 rank-r2 截断 SVD：

Code

```
[E_ℓ']_{r2} = Ũ_{r2} · Σ̃_{r2} · Ṽ_{r2}^T
```

#### 3.4 逆投影回原空间

将低秩近似映回原空间，得到补偿矩阵：

Code

```
W_ℓ^{comp} = [E_ℓ']_{r2} · Λ_ℓ^{-1/2} · Q_ℓ^T
           = Ũ_{r2} · Σ̃_{r2} · Ṽ_{r2}^T · Λ_ℓ^{-1/2} · Q_ℓ^T
```

#### 3.5 最终压缩权重

Code

```
W_ℓ' = W_ℓ^{(1)} + W_ℓ^{comp}
```

------

### 4. 理论分析

#### 4.1 预备引理

**Lemma A（Eckart-Young-Mirsky 定理）**

对任意矩阵 `M ∈ R^{m×n}`，其最优 rank-k 近似为截断 SVD `[M]_k`，且对任意 `rank(N) ≤ k`：

Code

```
||M - [M]_k||_F^2 ≤ ||M - N||_F^2
```

------

**Lemma B（特征空间等价性，源自 EoRA Theorem 1）**

设 `X·X^T = Q·Λ·Q^T`，`Q' = Q·Λ^{1/2}`，`Λ ≻ 0`。则对任意矩阵 `Δ` 和 `C`：

Code

```
||ΔX - CX||_F^2 = ||Δ·Q' - C·Q'||_F^2
```

**证明**：

由 Frobenius 范数与迹的关系：

Code

```
||ΔX - CX||_F^2 = trace((Δ-C)·XX^T·(Δ-C)^T)
                = trace((Δ-C)·Q·Λ·Q^T·(Δ-C)^T)
```

由 `Λ = Λ^{1/2}·Λ^{1/2}` 以及 `Q' = Q·Λ^{1/2}`：

Code

```
= trace(((Δ-C)·Q·Λ^{1/2})·((Δ-C)·Q·Λ^{1/2})^T)
= trace((Δ·Q' - C·Q')·(Δ·Q' - C·Q')^T)
= ||Δ·Q' - C·Q'||_F^2
```

证毕。■

进一步地，由于 `Q'` 列满秩（`Λ ≻ 0` 保证），变量替换 `C·Q' = S`，`rank(S) = rank(C)`，因此：

Code

```
min_{rank(C) ≤ r} ||ΔX - CX||_F^2 = min_{rank(S) ≤ r} ||Δ·Q' - S||_F^2
```

右边由 Lemma A 给出最优解 `S* = [Δ·Q']_r`，对应 `C* = S*·Q'^{-1} = [Δ·Q']_r·Λ^{-1/2}·Q^T`。

**这就是说：在特征空间做截断 SVD 后映回原空间，等价于直接最小化输出误差** `||ΔX - CX||_F^2`。■

------

#### 4.2 Lemma 1：残差补偿有效性

**引理陈述**：

> 设 Stage 1 得到 `W_ℓ^{(1)}`（rank ≤ r1），定义全局输出对齐误差 `E_ℓ = W_ℓ·X_ℓ^f - W_ℓ^{(1)}·X_ℓ`。Stage 2 通过特征空间方法求解：
>
> Code
>
> ```
> min_{rank(C) ≤ r2} ||E_ℓ - C·X_ℓ||_F^2
> ```
>
> 得到最优补偿 `W_ℓ^{comp}`。则：
>
> Code
>
> ```
> ||E_ℓ - W_ℓ^{comp}·X_ℓ||_F^2 ≤ ||E_ℓ||_F^2
> ```
>
> 即：加入 Stage 2 补偿后，全局输出对齐误差严格不增加。

**证明**：

Stage 2 的优化问题为：

Code

```
min_{rank(C) ≤ r2} ||E_ℓ - C·X_ℓ||_F^2
```

`W_ℓ^{comp}` 是该问题的最优解（由 Lemma B 保证，通过特征空间截断 SVD 求得）。

取 `C = 0`（零矩阵），有 `rank(0) = 0 ≤ r2`，是一个可行解。此时目标值为：

Code

```
||E_ℓ - 0·X_ℓ||_F^2 = ||E_ℓ||_F^2
```

由最优性：

Code

```
||E_ℓ - W_ℓ^{comp}·X_ℓ||_F^2 ≤ ||E_ℓ - 0·X_ℓ||_F^2 = ||E_ℓ||_F^2
```

证毕。■

**等价表述**（代回原始变量）：

Code

```
||W_ℓ·X_ℓ^f - (W_ℓ^{(1)} + W_ℓ^{comp})·X_ℓ||_F^2 ≤ ||W_ℓ·X_ℓ^f - W_ℓ^{(1)}·X_ℓ||_F^2
```

即：两阶段压缩结果 `W_ℓ'` 的输出与全精度参考输出之间的差距，不大于仅做 Stage 1 的差距。

------

#### 4.3 Corollary 1：特征空间补偿优于朴素权重空间补偿

**推论陈述**：

> 设 `W_ℓ^{naive}` 为对权重残差 `ΔW = W_ℓ - W_ℓ^{(1)}` 直接做 rank-r2 截断 SVD 的结果（不经过特征空间变换），即：
>
> Code
>
> ```
> W_ℓ^{naive} = [W_ℓ - W_ℓ^{(1)}]_{r2}
> ```
>
> 则：
>
> Code
>
> ```
> ||E_ℓ - W_ℓ^{comp}·X_ℓ||_F^2 ≤ ||E_ℓ - W_ℓ^{naive}·X_ℓ||_F^2
> ```

**证明**：

`W_ℓ^{naive}` 满足 `rank(W_ℓ^{naive}) = rank([W_ℓ - W_ℓ^{(1)}]_{r2}) ≤ r2`，因此是优化问题

Code

```
min_{rank(C) ≤ r2} ||E_ℓ - C·X_ℓ||_F^2
```

的一个**可行解**。

而 `W_ℓ^{comp}` 是该问题的**全局最优解**（由 Lemma B + Lemma A 保证）。

由最优性：

Code

```
||E_ℓ - W_ℓ^{comp}·X_ℓ||_F^2 ≤ ||E_ℓ - W_ℓ^{naive}·X_ℓ||_F^2
```

证毕。■

**物理意义**：朴素 SVD 补偿（如 ResSVD 的做法）最小化的是权重误差 `||ΔW - C||_F^2`，不等价于最小化输出误差 `||E_ℓ - C·X_ℓ||_F^2`（除非 `X_ℓ·X_ℓ^T = I`）。特征空间补偿通过考虑输入激活的分布，实现了对输出误差的直接最优化。

------

### 5. 整体理论总结

| 理论结果        | 结论                                  | 意义                                   |
| --------------- | ------------------------------------- | -------------------------------------- |
| Lemma B（预备） | 特征空间截断 SVD ⟺ 最小化输出误差     | 方法的正确性保证                       |
| Lemma 1         | `L_{补偿后} ≤ L_{补偿前}`             | Stage 2 不会"帮倒忙"，补偿严格有效     |
| Corollary 1     | `L_{特征空间补偿} ≤ L_{朴素权重补偿}` | 我们的补偿方式优于 ResSVD 式的朴素补偿 |

三者的逻辑链：

Code

```
Lemma B（等价性） → Lemma 1（有效性） → Corollary 1（优越性）
```

- Lemma B 保证方法在数学上是正确的（特征空间截断等价于最小化输出误差）
- Lemma 1 保证补偿有效（做了不会更差）
- Corollary 1 保证方法选择的优越性（比朴素做法更好）







# 创新点二（再次梳理）

### 1. 动机（Motivation）

#### 1.1 现有方法的不足

**问题一：权重空间补偿忽略激活分布**

ResSVD 等方法在权重空间最小化 `||W - W'||_F^2`，隐含假设所有方向的误差对输出影响相同。但实际上输入激活在不同方向上的能量分布极不均匀——某些方向的权重误差对输出影响很大，另一些几乎无影响。

**问题二：忽略上游累积误差**

逐层压缩时，当前层的输入 `X_ℓ` 已经偏离了全精度输入 `X_ℓ^f`。现有补偿方法（ResSVD、EoRA）仅考虑本层压缩误差 `(W_ℓ - W_ℓ^{(1)})·X_ℓ`，未将上游累积误差纳入补偿目标。

**问题三：Stage 1 截断后仍有残余误差**

即使 Stage 1 通过 Fisher-CEALC 做了主压缩，rank-r1 截断不可避免地丢弃了部分奇异值，仍存在未被消除的输出偏差。当压缩率较高时（r1 较小），这一残余误差尤为显著。

#### 1.2 我们的思路

在 Stage 1（Fisher-CEALC 主压缩，rank = r1）之后，利用额外的 rank-r2 预算，在特征空间对残余输出偏差做最优低秩补偿。总秩预算 `r = r1 + r2`。

------

### 2. 补偿目标的定义与合理性

#### 2.1 全局输出对齐误差

定义 Stage 1 压缩后的全局输出对齐误差：

$E_ℓ = W_ℓ · X_ℓ^f - W_ℓ^{(1)} · X_ℓ$



展开为两项：

$E_ℓ = W_ℓ·(X_ℓ^f - X_ℓ) + (W_ℓ - W_ℓ^{(1)})·X_ℓ$

| 项     | 公式                    | 含义                            |
| ------ | ----------------------- | ------------------------------- |
| 第一项 | $W_ℓ·(X_ℓ^f - X_ℓ)$     | 上游累积误差造成的输出偏差      |
| 第二项 | $(W_ℓ - W_ℓ^{(1)})·X_ℓ$ | 本层 Stage 1 截断造成的输出偏差 |

#### 2.2 与现有优化目标的关系

| 条件                        | E_ℓ 退化为                              | 对应论文      |
| --------------------------- | --------------------------------------- | ------------- |
| $X_ℓ^f = X_ℓ$（无累积误差） | $(W_ℓ - W_ℓ^{(1)})·X_ℓ$（标准激活误差） | SVD-LLM, EoRA |
| $X_ℓ = I$（单位输入）       | $W_ℓ·X_ℓ^f - W_ℓ^{(1)}$（近似权重误差） | ResSVD        |

`E_ℓ` 是标准激活误差的推广形式，与 SAES-SVD 的 CEALC 目标一脉相承：Stage 1 通过 β 平衡层内误差和累积误差做主压缩，Stage 2 对 Stage 1 未消除干净的总偏差做进一步补偿。

#### 2.3 Stage 2 的优化目标

Stage 2 求解：

$min_{rank(C) ≤ r2} ||E_ℓ - C · X_ℓ||_F^2$

即：在 rank ≤ r2 约束下，找到补偿矩阵 `C`，使 `C·X_ℓ` 尽可能逼近 `E_ℓ`。

补偿后第 ℓ 层的输出：

$W_ℓ' · X_ℓ = (W_ℓ^{(1)} + W_ℓ^{comp}) · X_ℓ ≈ W_ℓ · X_ℓ^f$

------

### 3. 方法详述

#### 3.1 特征分解与白化

对输入协方差矩阵做特征分解：

$H_ℓ = X_ℓ · X_ℓ^T = Q_ℓ · Λ_ℓ · Q_ℓ^T$

其中 `Q_ℓ` 为正交矩阵，`Λ_ℓ` 为特征值对角阵。定义白化投影矩阵：

$Q_ℓ' = Q_ℓ · Λ_ℓ^{1/2}$

注：需要对 `H_ℓ` 做阻尼正则化 $H_ℓ ← H_ℓ + λ_H·I $ 以保证正定性。

#### 3.2 投影到特征空间

将输出误差投影到特征空间：

$E_ℓ' = E_ℓ · Q_ℓ' = E_ℓ · Q_ℓ · Λ_ℓ^{1/2}$

#### 3.3 在特征空间做 rank-r2 截断 SVD

对 `E_ℓ'` 做 rank-r2 截断 SVD：

$[E_ℓ']_{r2} = Ũ_{r2} · Σ̃_{r2} · Ṽ_{r2}^T$

#### 3.4 逆投影回原空间

将低秩近似映回原空间，得到补偿矩阵：

$W_ℓ^{comp} = [E_ℓ']_{r2} · Λ_ℓ^{-1/2} · Q_ℓ^T$

​       $= Ũ_{r2} · Σ̃_{r2} · Ṽ_{r2}^T · Λ_ℓ^{-1/2} · Q_ℓ^T$

对应的低秩因子分解：

$B_ℓ' = Ũ_{r2} · Σ̃_{r2}^{1/2}                          ∈ R^{m × r2}$
$A_ℓ' = Σ̃_{r2}^{1/2} · Ṽ_{r2}^T · Λ_ℓ^{-1/2} · Q_ℓ^T  ∈ R^{r2 × n}$

验证：

$B_ℓ' · A_ℓ' = Ũ_{r2} · Σ̃_{r2} · Ṽ_{r2}^T · Λ_ℓ^{-1/2} · Q_ℓ^T = W_ℓ^{comp}  ✓$

#### 3.5 最终压缩权重

$W_ℓ' = W_ℓ^{(1)} + W_ℓ^{comp}$

最终第 ℓ 层的前向传播：

$output = W_ℓ^{(1)} · X_ℓ + W_ℓ^{comp} · X_ℓ$

​       $= A_ℓ^{(1)} · B_ℓ^{(1)} · X_ℓ + B_ℓ' · A_ℓ' · X_ℓ$

------

### 4. 理论分析

#### 4.1 预备引理

**Lemma A（Eckart-Young-Mirsky 定理）**

对任意矩阵 `M ∈ R^{m×n}`，其最优 rank-k 近似为截断 SVD `[M]_k`，且对任意 `rank(N) ≤ k`：

$||M - [M]_k||_F^2 ≤ ||M - N||_F^2$

------

**Lemma B（特征空间等价性，源自 EoRA Theorem 1）**

设 `X·X^T = Q·Λ·Q^T`，`Q' = Q·Λ^{1/2}`，`Λ ≻ 0`。则对任意矩阵 `Δ` 和 `C`：

$||ΔX - CX||_F^2 = ||Δ·Q' - C·Q'||_F^2$

**证明**：

Code

```
||ΔX - CX||_F^2
= trace((Δ-C) · XX^T · (Δ-C)^T)
= trace((Δ-C) · Q·Λ·Q^T · (Δ-C)^T)
```

由 `Λ = Λ^{1/2}·Λ^{1/2}` 以及 `Q' = Q·Λ^{1/2}`：

Code

```
= trace(((Δ-C)·Q·Λ^{1/2}) · ((Δ-C)·Q·Λ^{1/2})^T)
= ||(Δ-C)·Q'||_F^2
= ||Δ·Q' - C·Q'||_F^2
```

证毕。■

**推论**：由于 `Q'` 列满秩（`Λ ≻ 0` 保证），变量替换 `S = C·Q'`，`rank(S) = rank(C)`，因此：

Code

```
min_{rank(C) ≤ r} ||ΔX - CX||_F^2 = min_{rank(S) ≤ r} ||Δ·Q' - S||_F^2
```

右边由 Lemma A 给出最优解 `S* = [Δ·Q']_r`，对应：

Code

```
C* = S*·Q'^{-1} = [Δ·Q']_r · Λ^{-1/2} · Q^T
```

**即：在特征空间做截断 SVD 后映回原空间，等价于直接最小化输出误差。** ■

------

#### 4.2 Lemma 1：残差补偿有效性

**引理陈述**：

> 设 Stage 1 得到 `W_ℓ^{(1)}`（rank ≤ r1），定义全局输出对齐误差：
>
> Code
>
> ```
> E_ℓ = W_ℓ · X_ℓ^f - W_ℓ^{(1)} · X_ℓ
> ```
>
> Stage 2 通过特征空间方法求解：
>
> Code
>
> ```
> min_{rank(C) ≤ r2} ||E_ℓ - C·X_ℓ||_F^2
> ```
>
> 得到最优补偿 `W_ℓ^{comp}`。则：
>
> Code
>
> ```
> ||E_ℓ - W_ℓ^{comp}·X_ℓ||_F^2 ≤ ||E_ℓ||_F^2
> ```
>
> 即：Stage 2 补偿后，全局输出对齐误差严格不增加。

**证明**：

Stage 2 的优化问题为：

Code

```
min_{rank(C) ≤ r2} ||E_ℓ - C·X_ℓ||_F^2
```

`W_ℓ^{comp}` 是该问题的全局最优解（由 Lemma B + Lemma A 保证）。

取 `C = 0`（零矩阵），有 `rank(0) = 0 ≤ r2`，是一个可行解。此时目标值为：

Code

```
||E_ℓ - 0·X_ℓ||_F^2 = ||E_ℓ||_F^2
```

由最优性：

Code

```
||E_ℓ - W_ℓ^{comp}·X_ℓ||_F^2 ≤ ||E_ℓ||_F^2
```

证毕。■

**等价表述**（代回原始变量）：

Code

```
||W_ℓ·X_ℓ^f - (W_ℓ^{(1)} + W_ℓ^{comp})·X_ℓ||_F^2 ≤ ||W_ℓ·X_ℓ^f - W_ℓ^{(1)}·X_ℓ||_F^2
```

------

#### 4.3 Corollary 1：特征空间补偿优于朴素权重空间补偿

**推论陈述**：

> 设 `W_ℓ^{naive}` 为对权重残差 `ΔW = W_ℓ - W_ℓ^{(1)}` 直接做 rank-r2 截断 SVD 的结果（不经过特征空间变换）：
>
> Code
>
> ```
> W_ℓ^{naive} = [W_ℓ - W_ℓ^{(1)}]_{r2}
> ```
>
> 则：
>
> Code
>
> ```
> ||E_ℓ - W_ℓ^{comp}·X_ℓ||_F^2 ≤ ||E_ℓ - W_ℓ^{naive}·X_ℓ||_F^2
> ```

**证明**：

`W_ℓ^{naive}` 满足 `rank(W_ℓ^{naive}) ≤ r2`，因此是优化问题

Code

```
min_{rank(C) ≤ r2} ||E_ℓ - C·X_ℓ||_F^2
```

的一个可行解。

`W_ℓ^{comp}` 是该问题的全局最优解。

由最优性：

Code

```
||E_ℓ - W_ℓ^{comp}·X_ℓ||_F^2 ≤ ||E_ℓ - W_ℓ^{naive}·X_ℓ||_F^2
```

证毕。■

**物理意义**：朴素 SVD 补偿最小化的是权重误差 `||ΔW - C||_F^2`，不等价于最小化输出误差 `||E_ℓ - C·X_ℓ||_F^2`（除非 `X_ℓ·X_ℓ^T = I`）。特征空间补偿通过考虑输入激活的分布，实现了对输出误差的直接最优化。
