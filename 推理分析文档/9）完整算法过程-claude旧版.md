## 完整算法及验算流程：FR-SVD（Fisher-Residual SVD）

### Phase 0：数据准备

**Step 0.1**：用原始模型前向传播校准数据，逐层记录全精度激活 `{X_ℓ^f}_{ℓ=1}^L`

**Step 0.2**：初始化压缩激活 `X_1 ← X_1^f`（第一层输入无累积误差）

### Phase 1：逐层压缩（for ℓ = 1 to L）

---

#### Stage 1：Fisher-CEALC 主压缩

---

**Step 1.1**：构造二阶统计量

```
H_ℓ = X_ℓ · X_ℓ^T + λ_H · I      ∈ R^{n×n}   （阻尼输入协方差）
Δ_ℓ = (X_ℓ^f - X_ℓ) · X_ℓ^T      ∈ R^{n×n}   （累积误差协方差）
```

**维度验算**：
- `X_ℓ ∈ R^{n×p}`（n = 输入维度，p = 样本数）
- `X_ℓ^f ∈ R^{n×p}`
- `X_ℓ · X_ℓ^T ∈ R^{n×n}` ✓
- `(X_ℓ^f - X_ℓ) · X_ℓ^T ∈ R^{n×n}` ✓

**注意**：第一层 `X_1 = X_1^f`，所以 `Δ_1 = 0`，`β_1* = 0`。

---

**Step 1.2**：逐层 Fisher 信息估计

```
对每个 batch_i ∈ D:
    G_i = ∇_{W_ℓ} L(batch_i)     ∈ R^{m×n}
    g_i = vec(G_i)                ∈ R^{mn}

I_F^{(ℓ)} = (1/|D|) · Σ_{i=1}^{|D|} g_i · g_i^T    ∈ R^{mn × mn}
```

**维度验算**：
- `W_ℓ ∈ R^{m×n}`
- `G_i ∈ R^{m×n}`，`g_i ∈ R^{mn}`
- `g_i · g_i^T ∈ R^{mn × mn}` ✓

---

**Step 1.3**：Kronecker 分解

```
I_F^{(ℓ)} ≈ A_ℓ ⊗ B_ℓ
```

其中 `A_ℓ ∈ R^{n×n}`，`B_ℓ ∈ R^{m×m}`。

**验算**：`A_ℓ ⊗ B_ℓ ∈ R^{mn × mn}` = `I_F^{(ℓ)}` 的维度 ✓

具体方法（GFWSVD Algorithm 1）：

```
Ĩ_F = R(I_F^{(ℓ)})                       （置换操作，R^{m²×n²}）
(u, σ, v^T) ← 截断SVD的leading triplet
b = u · σ → reshape → B_ℓ ∈ R^{m×m}
a = v     → reshape → A_ℓ ∈ R^{n×n}
```

高效计算（GFWSVD Eq.15）：不显式构造 `Ĩ_F`，利用：

```
Ĩ_F · z = (1/|D|) · Σ_{i=1}^{|D|} vec(G_i · Z · G_i^T)
```

其中 `z = vec(Z)`，`Z ∈ R^{n×n}`，输出 `∈ R^{m²}`。✓

---

**Step 1.4**：Cholesky 分解

```
A_ℓ + λ_A · I = L_A^{(ℓ)} · L_A^{(ℓ)T}     L_A^{(ℓ)} ∈ R^{n×n}（下三角）
B_ℓ + λ_B · I = L_B^{(ℓ)} · L_B^{(ℓ)T}     L_B^{(ℓ)} ∈ R^{m×m}（下三角）
```

**验算**：`L_A · L_A^T ∈ R^{n×n}` = `A_ℓ` 的维度 ✓，`L_B · L_B^T ∈ R^{m×m}` = `B_ℓ` 的维度 ✓

---

**Step 1.5**：构造 CEALC 目标矩阵

```
H_ℓ^{-1/2} ∈ R^{n×n}                    （H_ℓ 的逆平方根）
G_ℓ(β) = W_ℓ · (H_ℓ + β·Δ_ℓ) · H_ℓ^{-1/2}
```

**维度验算**：
- `W_ℓ ∈ R^{m×n}`
- `H_ℓ + β·Δ_ℓ ∈ R^{n×n}`
- `W_ℓ · (H_ℓ + β·Δ_ℓ) ∈ R^{m×n}`
- `· H_ℓ^{-1/2} ∈ R^{n×n}`
- `G_ℓ(β) ∈ R^{m×n}` ✓

分解为基底和扰动：

```
S_ℓ = W_ℓ · H_ℓ · H_ℓ^{-1/2} = W_ℓ · H_ℓ^{1/2}    ∈ R^{m×n}
D_ℓ = W_ℓ · Δ_ℓ · H_ℓ^{-1/2}                        ∈ R^{m×n}
G_ℓ(β) = S_ℓ + β · D_ℓ                               ∈ R^{m×n}
```

**验算**：
- `W_ℓ · H_ℓ^{1/2} ∈ R^{m×n} × R^{n×n} = R^{m×n}` ✓
- `W_ℓ · Δ_ℓ · H_ℓ^{-1/2} ∈ R^{m×n} × R^{n×n} × R^{n×n} = R^{m×n}` ✓

---

**Step 1.6**：变换到 Fisher 度量空间

```
S_ℓ^F = L_B^T · S_ℓ · L_A    ∈ R^{m×n}
D_ℓ^F = L_B^T · D_ℓ · L_A    ∈ R^{m×n}
G_ℓ^F(β) = S_ℓ^F + β · D_ℓ^F = L_B^T · G_ℓ(β) · L_A    ∈ R^{m×n}
```

**维度验算**：
- `L_B^T ∈ R^{m×m}`，`S_ℓ ∈ R^{m×n}`，`L_A ∈ R^{n×n}`
- `L_B^T · S_ℓ ∈ R^{m×n}`
- `· L_A ∈ R^{m×n}` ✓

**验算线性性**：

```
L_B^T · G_ℓ(β) · L_A = L_B^T · (S_ℓ + β·D_ℓ) · L_A
                      = L_B^T · S_ℓ · L_A + β · L_B^T · D_ℓ · L_A
                      = S_ℓ^F + β · D_ℓ^F  ✓
```

---

**Step 1.7**：在 Fisher 空间中优化 β

对 `S_ℓ^F` 做 rank-r1 的 SVD：`S_ℓ^F = U_S · Σ_S · V_S^T`

构造正交补投影矩阵：

```
P_L = I_m - U_{r1} · U_{r1}^T    ∈ R^{m×m}
P_R = I_n - V_{r1} · V_{r1}^T    ∈ R^{n×n}
```

其中 `U_{r1}` 是 `S_ℓ^F` 的 top-r1 左奇异向量，`V_{r1}` 是 top-r1 右奇异向量。

计算正交补分量：

```
S_ℓ^{F⊥} = P_L · S_ℓ^F · P_R    ∈ R^{m×n}
D_ℓ^{F⊥} = P_L · D_ℓ^F · P_R    ∈ R^{m×n}
```

计算六个标量：

```
a = ||S_ℓ^{F⊥}||_F^2           A = ||S_ℓ^F||_F^2
b = ⟨S_ℓ^{F⊥}, D_ℓ^{F⊥}⟩      B = ⟨S_ℓ^F, D_ℓ^F⟩
c = ||D_ℓ^{F⊥}||_F^2           C = ||D_ℓ^F||_F^2
```

其中 `⟨A, B⟩ = trace(A^T · B)`。

近似误差比：

```
ρ̃(β) = (a + 2bβ + cβ^2) / (A + 2Bβ + Cβ^2)
```

对 `ρ̃(β)` 求导令其为零，得到二次方程：

```
(cB - bC)·β^2 + (cA - aC)·β + (bA - aB) = 0
```

**验算求导过程**：

```
ρ̃(β) = f(β) / g(β)

其中 f = a + 2bβ + cβ^2，g = A + 2Bβ + Cβ^2

ρ̃'(β) = (f'g - fg') / g^2 = 0
→ f'g - fg' = 0

f' = 2b + 2cβ
g' = 2B + 2Cβ

f'g - fg' = (2b + 2cβ)(A + 2Bβ + Cβ^2) - (a + 2bβ + cβ^2)(2B + 2Cβ)
```

展开 `f'g`：

```
= 2bA + 4bBβ + 2bCβ^2 + 2cAβ + 4cBβ^2 + 2cCβ^3
```

展开 `fg'`：

```
= 2aB + 2aCβ + 4bBβ + 4bCβ^2 + 2cBβ^2 + 2cCβ^3
```

相减：

```
f'g - fg' = (2bA - 2aB) + (4bB + 2cA - 2aC - 4bB)β + (2bC + 4cB - 4bC - 2cB)β^2 + (2cC - 2cC)β^3
           = (2bA - 2aB) + (2cA - 2aC)β + (4cB - 2bC - 2cB)β^2... 
```

**重新仔细展开**：

```
f'g = (2b + 2cβ)(A + 2Bβ + Cβ^2)
    = 2bA + 4bBβ + 2bCβ^2 + 2cAβ + 4cBβ^2 + 2cCβ^3

fg' = (a + 2bβ + cβ^2)(2B + 2Cβ)
    = 2aB + 2aCβ + 4bBβ + 4bCβ^2 + 2cBβ^2 + 2cCβ^3

f'g - fg' = (2bA - 2aB) + (4bB + 2cA - 2aC - 4bB)β + (2bC + 4cB - 4bC - 2cB)β^2
           = 2(bA - aB) + 2(cA - aC)β + 2(4cB - 4bC + 2bC - 2cB)β^2 / 2...
```

让我更仔细：

β^0 项：`2bA - 2aB`
β^1 项：`4bB + 2cA - 2aC - 4bB = 2cA - 2aC = 2(cA - aC)`
β^2 项：`2bC + 4cB - 4bC - 2cB = 2(2cB - cB) + 2(bC - 2bC) = 2cB - 2bC = 2(cB - bC)`
β^3 项：`2cC - 2cC = 0`

所以：

```
f'g - fg' = 2(bA - aB) + 2(cA - aC)β + 2(cB - bC)β^2 = 0
```

除以 2：

```
(cB - bC)β^2 + (cA - aC)β + (bA - aB) = 0  ✓
```

与 SAES-SVD Eq.19 一致。✓

取 `[0, 1]` 内使 `ρ̃(β)` 最小（即保留能量比最大）的根为 `β*`。

**Step 1.8**：在 Fisher 空间做 rank-r1 截断 SVD

```
G_ℓ^F(β*) = S_ℓ^F + β* · D_ℓ^F    ∈ R^{m×n}

[G_ℓ^F(β*)]_{r1} = Û_{r1} · Ŝ_{r1} · V̂_{r1}^T
```

其中：
- `Û_{r1} ∈ R^{m×r1}`
- `Ŝ_{r1} ∈ R^{r1×r1}`（对角）
- `V̂_{r1} ∈ R^{n×r1}`

**Step 1.9**：映回原空间得到压缩权重

映回 G 空间：

```
T_ℓ* = L_B^{-T} · Û_{r1} · Ŝ_{r1} · V̂_{r1}^T · L_A^{-1}    ∈ R^{m×n}
```

**维度验算**：
- `L_B^{-T} ∈ R^{m×m}`，`Û_{r1} ∈ R^{m×r1}` → `L_B^{-T} · Û_{r1} ∈ R^{m×r1}`
- `· Ŝ_{r1} ∈ R^{r1×r1}` → `R^{m×r1}`
- `· V̂_{r1}^T ∈ R^{r1×n}` → `R^{m×n}`
- `· L_A^{-1} ∈ R^{n×n}` → `R^{m×n}` ✓

**验算 T_ℓ* 确实是 J(T) 的最优解**：

```
J(T) = ||L_B^T · (G_ℓ(β*) - T) · L_A||_F^2

令 S = L_B^T · T · L_A，则 J(T) = ||L_B^T · G_ℓ(β*) · L_A - S||_F^2 = ||G_ℓ^F(β*) - S||_F^2

最优 S* = [G_ℓ^F(β*)]_{r1}
→ T* = L_B^{-T} · S* · L_A^{-1} = L_B^{-T} · [G_ℓ^F(β*)]_{r1} · L_A^{-1}  ✓
```

映回权重空间：

```
W_ℓ^{(1)} = T_ℓ* · H_ℓ^{-1/2}
           = L_B^{-T} · Û_{r1} · Ŝ_{r1} · V̂_{r1}^T · L_A^{-1} · H_ℓ^{-1/2}    ∈ R^{m×n}
```

**维度验算**：`R^{m×n} · R^{n×n} = R^{m×n}` ✓

**验算为什么乘 H_ℓ^{-1/2}**：

SAES-SVD 的等价变换（Theorem 4.1）将原始优化问题

```
min ||A·B·X_ℓ - Z_ℓ||_F^2
```

变换为

```
min ||A·B·H^{1/2} - G(β)||_F^2
```

令 `T = A·B·H^{1/2}`，则 `A·B = T·H^{-1/2}`，即压缩权重 `W' = T·H^{-1/2}`。✓

低秩因子：

```
A_ℓ^{(1)} = L_B^{-T} · Û_{r1} · Ŝ_{r1}^{1/2}                    ∈ R^{m×r1}
B_ℓ^{(1)} = Ŝ_{r1}^{1/2} · V̂_{r1}^T · L_A^{-1} · H_ℓ^{-1/2}    ∈ R^{r1×n}
```

**验算乘积**：

```
A_ℓ^{(1)} · B_ℓ^{(1)} = L_B^{-T} · Û_{r1} · Ŝ_{r1}^{1/2} · Ŝ_{r1}^{1/2} · V̂_{r1}^T · L_A^{-1} · H_ℓ^{-1/2}
                       = L_B^{-T} · Û_{r1} · Ŝ_{r1} · V̂_{r1}^T · L_A^{-1} · H_ℓ^{-1/2}
                       = W_ℓ^{(1)}  ✓
```

**验算秩**：`rank(W_ℓ^{(1)}) = rank(Û_{r1} · Ŝ_{r1} · V̂_{r1}^T) ≤ r1` ✓（左右乘可逆矩阵不改变秩）

#### Stage 2：特征空间残差补偿

---

**Step 2.1**：计算全局输出对齐误差

```
E_ℓ = W_ℓ · X_ℓ^f - W_ℓ^{(1)} · X_ℓ    ∈ R^{m×p}
```

**维度验算**：
- `W_ℓ ∈ R^{m×n}`，`X_ℓ^f ∈ R^{n×p}` → `W_ℓ · X_ℓ^f ∈ R^{m×p}`
- `W_ℓ^{(1)} ∈ R^{m×n}`，`X_ℓ ∈ R^{n×p}` → `W_ℓ^{(1)} · X_ℓ ∈ R^{m×p}`
- `E_ℓ ∈ R^{m×p}` ✓

---

**Step 2.2**：求无秩约束的最小二乘解

目标：

```
min_{ΔW} ||ΔW · X_ℓ - E_ℓ||_F^2 + λ||ΔW||_F^2
```

闭式解：

```
ΔW_ℓ^{LS} = E_ℓ · X_ℓ^T · H_ℓ^{-1}    ∈ R^{m×n}
```

其中 `H_ℓ = X_ℓ · X_ℓ^T + λ_H · I ∈ R^{n×n}`。

**维度验算**：
- `E_ℓ ∈ R^{m×p}`，`X_ℓ^T ∈ R^{p×n}` → `E_ℓ · X_ℓ^T ∈ R^{m×n}`
- `· H_ℓ^{-1} ∈ R^{n×n}` → `ΔW_ℓ^{LS} ∈ R^{m×n}` ✓

**验算闭式解的推导**：

```
J(ΔW) = trace((ΔW·X - E)·(ΔW·X - E)^T) + λ·trace(ΔW·ΔW^T)

∂J/∂ΔW = 2·ΔW·X·X^T - 2·E·X^T + 2λ·ΔW = 0

→ ΔW·(X·X^T + λI) = E·X^T

→ ΔW = E·X^T·(X·X^T + λI)^{-1} = E·X^T·H^{-1}  ✓
```

**验算 ΔW_ℓ^{LS}·X_ℓ 的拟合效果**：

```
ΔW_ℓ^{LS} · X_ℓ = E_ℓ · X_ℓ^T · H_ℓ^{-1} · X_ℓ
```

当 `λ → 0`，`H_ℓ^{-1} ≈ (X·X^T)^{-1}`：

```
= E_ℓ · X_ℓ^T · (X_ℓ·X_ℓ^T)^{-1} · X_ℓ = E_ℓ · P
```

其中 `P = X_ℓ^T·(X_ℓ·X_ℓ^T)^{-1}·X_ℓ` 是 `X_ℓ` 列空间上的正交投影矩阵。即 `ΔW^{LS}·X_ℓ` 是 `E_ℓ` 在 `X_ℓ` 列空间上的投影。✓

---

**Step 2.3**：特征空间白化

对 `H_ℓ` 做特征分解：

```
H_ℓ = Q_ℓ · Λ_ℓ · Q_ℓ^T
```

其中 `Q_ℓ ∈ R^{n×n}`（正交），`Λ_ℓ ∈ R^{n×n}`（对角正定）。

白化变换：

```
ΔW_{white} = ΔW_ℓ^{LS} · Q_ℓ · Λ_ℓ^{1/2}    ∈ R^{m×n}
```

**维度验算**：
- `ΔW_ℓ^{LS} ∈ R^{m×n}`，`Q_ℓ ∈ R^{n×n}`，`Λ_ℓ^{1/2} ∈ R^{n×n}`
- `ΔW_{white} ∈ R^{m×n}` ✓

**验算白化的意义**：

Stage 2 要最优化的实际目标是：

```
min_{rank(C)≤r2} ||C·X_ℓ - ΔW_ℓ^{LS}·X_ℓ||_F^2 = min_{rank(C)≤r2} ||(ΔW_ℓ^{LS} - C)·X_ℓ||_F^2
```

由 Frobenius 范数与迹的关系：

```
||(ΔW^{LS} - C)·X||_F^2 = trace((ΔW^{LS} - C)·XX^T·(ΔW^{LS} - C)^T)
                         = trace((ΔW^{LS} - C)·QΛQ^T·(ΔW^{LS} - C)^T)
                         = ||(ΔW^{LS} - C)·Q·Λ^{1/2}||_F^2
```

令 `S = C·Q·Λ^{1/2}`，`rank(S) = rank(C)`：

```
= ||ΔW^{LS}·Q·Λ^{1/2} - S||_F^2 = ||ΔW_{white} - S||_F^2
```

由 Eckart-Young-Mirsky 定理，最优 `S* = [ΔW_{white}]_{r2}`。✓

---

**Step 2.4**：rank-r2 截断 SVD

```
[ΔW_{white}]_{r2} = Ũ_{r2} · Σ̃_{r2} · Ṽ_{r2}^T
```

其中：
- `Ũ_{r2} ∈ R^{m×r2}`
- `Σ̃_{r2} ∈ R^{r2×r2}`（对角）
- `Ṽ_{r2} ∈ R^{n×r2}`

---

**Step 2.5**：逆投影回原空间

```
W_ℓ^{comp} = [ΔW_{white}]_{r2} · Λ_ℓ^{-1/2} · Q_ℓ^T    ∈ R^{m×n}
           = Ũ_{r2} · Σ̃_{r2} · Ṽ_{r2}^T · Λ_ℓ^{-1/2} · Q_ℓ^T
```

**维度验算**：
- `[ΔW_{white}]_{r2} ∈ R^{m×n}`，`Λ_ℓ^{-1/2} ∈ R^{n×n}`，`Q_ℓ^T ∈ R^{n×n}`
- `W_ℓ^{comp} ∈ R^{m×n}` ✓

**验算映回过程**：

由 `S* = C*·Q·Λ^{1/2} = [ΔW_{white}]_{r2}`：

```
C* = S*·(Q·Λ^{1/2})^{-1} = [ΔW_{white}]_{r2}·Λ^{-1/2}·Q^T  ✓
```

（因为 `(Q·Λ^{1/2})^{-1} = Λ^{-1/2}·Q^{-1} = Λ^{-1/2}·Q^T`）✓

**验算秩**：`rank(W_ℓ^{comp}) = rank([ΔW_{white}]_{r2}) ≤ r2` ✓（右乘可逆矩阵不改变秩）

低秩因子：

```
B_ℓ' = Ũ_{r2} · Σ̃_{r2}^{1/2}                              ∈ R^{m×r2}
A_ℓ' = Σ̃_{r2}^{1/2} · Ṽ_{r2}^T · Λ_ℓ^{-1/2} · Q_ℓ^T      ∈ R^{r2×n}
```

**验算乘积**：

```
B_ℓ' · A_ℓ' = Ũ_{r2}·Σ̃_{r2}^{1/2}·Σ̃_{r2}^{1/2}·Ṽ_{r2}^T·Λ^{-1/2}·Q^T
             = Ũ_{r2}·Σ̃_{r2}·Ṽ_{r2}^T·Λ^{-1/2}·Q^T
             = W_ℓ^{comp}  ✓
```

---

**Step 2.6**：组合最终压缩权重

```
W_ℓ' = W_ℓ^{(1)} + W_ℓ^{comp}    ∈ R^{m×n}
```

**验算总秩**：`rank(W_ℓ') ≤ rank(W_ℓ^{(1)}) + rank(W_ℓ^{comp}) ≤ r1 + r2 = r` ✓

最终前向传播表示为：

```
output_ℓ = W_ℓ' · X_ℓ = A_ℓ^{(1)}·B_ℓ^{(1)}·X_ℓ + B_ℓ'·A_ℓ'·X_ℓ
```

| `G_ℓ^F(β)`   | `m × n`            | Fisher 空间目标矩阵      |
| `W_ℓ^{(1)}`  | `m × n`，rank ≤ r1 | Stage 1 压缩权重         |
| `E_ℓ`        | `m × p`            | 全局输出对齐误差         |
| `ΔW_ℓ^{LS}`  | `m × n`            | 最小二乘解（满秩）       |
| `ΔW_{white}` | `m × n`            | 白化后的最小二乘解       |
| `W_ℓ^{comp}` | `m × n`，rank ≤ r2 | 补偿矩阵                 |
| `W_ℓ'`       | `m × n`，rank ≤ r  | 最终压缩权重             |

## 完整算法伪代码

```
Algorithm: FR-SVD (Fisher-Residual SVD)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Input:  原始模型 {W_ℓ}，校准集 D，压缩比 R，rank分配 r1:r2
Output: 压缩模型 {W_ℓ'}

// Phase 0: 数据准备
1:  {X_ℓ^f} ← 原始模型前向传播 D
2:  X_1 ← X_1^f

// Phase 1: 逐层压缩
3:  for ℓ = 1 to L do

      // === Stage 1: Fisher-CEALC 主压缩 ===
4:    H_ℓ ← X_ℓ·X_ℓ^T + λ_H·I
5:    Δ_ℓ ← (X_ℓ^f - X_ℓ)·X_ℓ^T
6:    {G_i} ← 计算第ℓ层梯度
7:    A_ℓ, B_ℓ ← KroneckerDecompose({G_i})
8:    L_A ← Cholesky(A_ℓ + λ_A·I)
9:    L_B ← Cholesky(B_ℓ + λ_B·I)
10:   S_ℓ^F ← L_B^T · W_ℓ · H_ℓ^{1/2} · L_A
11:   D_ℓ^F ← L_B^T · W_ℓ · Δ_ℓ · H_ℓ^{-1/2} · L_A
12:   β* ← 求解 FS-FOA 二次方程（基于 S_ℓ^F, D_ℓ^F）
13:   G_ℓ^F(β*) ← S_ℓ^F + β*·D_ℓ^F
14:   Û, Ŝ, V̂^T ← SVD_{r1}(G_ℓ^F(β*))
15:   W_ℓ^{(1)} ← L_B^{-T}·Û·Ŝ·V̂^T·L_A^{-1}·H_ℓ^{-1/2}

      // === Stage 2: 特征空间残差补偿 ===
16:   E_ℓ ← W_ℓ·X_ℓ^f - W_ℓ^{(1)}·X_ℓ
17:   ΔW^{LS} ← E_ℓ·X_ℓ^T·H_ℓ^{-1}
18:   Q, Λ ← Eigen(H_ℓ)
19:   ΔW_{white} ← ΔW^{LS}·Q·Λ^{1/2}
20:   Ũ, Σ̃, Ṽ^T ← SVD_{r2}(ΔW_{white})
21:   W_ℓ^{comp} ← Ũ·Σ̃·Ṽ^T·Λ^{-1/2}·Q^T

      // === 组合 + 传播 ===
22:   W_ℓ' ← W_ℓ^{(1)} + W_ℓ^{comp}
23:   X_{ℓ+1} ← ForwardPass(W_ℓ', X_ℓ)

24: end for
25: return {W_ℓ'}
```







## FR-SVD 完整算法step流程

------

### Phase 0：数据准备

Code

```
输入：原始模型权重 {W_ℓ}，校准集 D，rank 分配 r1, r2（r = r1 + r2）
1:  {X_ℓ^f}_{ℓ=1}^L ← 原始模型前向传播 D，逐层记录全精度激活
2:  X_1 ← X_1^f
```

------

### Phase 1：逐层压缩（for ℓ = 1 to L）

#### Stage 1：Fisher-CEALC 主压缩

Code

```
 3:  H_ℓ = X_ℓ·X_ℓ^T + λ_H·I                              // 阻尼输入协方差 (n×n)
 4:  Δ_ℓ = (X_ℓ^f - X_ℓ)·X_ℓ^T                             // 累积误差协方差 (n×n)
 5:  A_ℓ ⊗ B_ℓ ← KroneckerDecompose(∇_{W_ℓ} L)            // Fisher Kronecker 分解
 6:  L_A ← Cholesky(A_ℓ + λ_A·I)                           // (n×n)
 7:  L_B ← Cholesky(B_ℓ + λ_B·I)                           // (m×m)
 8:  S_ℓ^F = L_B^T · W_ℓ · H_ℓ^{1/2} · L_A                // Fisher 空间基底 (m×n)
 9:  D_ℓ^F = L_B^T · W_ℓ · Δ_ℓ · H_ℓ^{-1/2} · L_A        // Fisher 空间扰动 (m×n)
10:  β* ← 求解 FS-FOA 二次方程（基于 S_ℓ^F, D_ℓ^F）        // 最优对齐系数
11:  Û, Ŝ, V̂^T ← SVD_{r1}(S_ℓ^F + β*·D_ℓ^F)              // Fisher 空间截断
12:  W_ℓ^{(1)} = L_B^{-T}·Û·Ŝ·V̂^T·L_A^{-1}·H_ℓ^{-1/2}   // 映回权重空间 (m×n, rank≤r1)
```

#### Stage 2：特征空间残差补偿

Code

```
13:  E_ℓ = W_ℓ·X_ℓ^f - W_ℓ^{(1)}·X_ℓ                      // 全局输出对齐误差 (m×p)
14:  ΔW^{LS} = E_ℓ·X_ℓ^T·H_ℓ^{-1}                         // 最小二乘解 (m×n)
15:  Q, Λ ← Eigen(H_ℓ)                                     // 特征分解（可复用 Step 3）
16:  ΔW_{white} = ΔW^{LS}·Q·Λ^{1/2}                        // 白化 (m×n)
17:  Ũ, Σ̃, Ṽ^T ← SVD_{r2}(ΔW_{white})                     // 特征空间截断
18:  W_ℓ^{comp} = Ũ·Σ̃·Ṽ^T·Λ^{-1/2}·Q^T                   // 映回原空间 (m×n, rank≤r2)
```

#### 组合 + 传播

Code

```
19:  W_ℓ' = W_ℓ^{(1)} + W_ℓ^{comp}                         // 最终权重 (m×n, rank≤r)
20:  X_{ℓ+1} ← ForwardPass(W_ℓ', X_ℓ)                     // 传播到下一层
```

```
end for
输出：压缩模型 {W_ℓ'}
```
